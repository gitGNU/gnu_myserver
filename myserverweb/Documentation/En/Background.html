MYSERVER PROJECT

MyServer is a free and easily configurable web server available for both the
Windows and GNU/Linux Operating Systems.


MyServer's principal goal is to create a free, and  powerful application to allow anyone to transform his or her  home PC into a web server and share web files easily.  As it is a multithread application that supports multiprocessor machines, it can also  be used professionally.

Please visit our site http://www.myserverproject.net for more information and to get the latest news about this project.

You can use the site forum to post any questions or comments you may have about the project.
