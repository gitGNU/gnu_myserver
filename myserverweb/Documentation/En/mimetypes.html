MIME TYPES

You can modify the MIME types used by MyServer by modifing the file "MIMEtypes.txt" with a text application such as notepad in Windows or Emacs in Linux. Each line describe a different MIME type. For example, the following line is used for the files with .html extension:

html,text/html,SEND NONE;


This line indicates the extension to register (html), the MIME type to be used (text/html), the way to send the file to the client (SEND), and finally, an optional path used to fully describe the previous command (NONE)


Valid values for sending the file to the client are:

  1. SEND - Send the file statically as is.  This command is used for files extensions such as .html and .jpg that don't need any external intervention from the web server.
  2. RUNCGI - Run the CGI script indicated in the path.  It is used for file extensions such as .php and .pl that need to be processed before sending any data to the client.
  3. RUNFASTCGI - Run the Fastcgi script.  This type will need a fastcgi parser downloaded (see the PHP page).  It is also used for file extensions such as .php and .pl
  4. EXECUTEFASTCGI - This also runs the Fastcgi script, but unlike the RUNFASTCGI above, it does not need a server to run the script, instead it acts as a self server.
  5. RUNMSCGI - Run the MSCGI script.  This library is only available for myServer.  It is used to build fast and efficient CGIs.
  6. EXECUTE - Treat the file as an executable and send the output to the client. 
  7. SENDLINK - Send the file that is contained in a link.
  8. EXECUTEWINCGI - Send the file as a WinCGI
  9. RUNISAPI - Run the ISAPI script.  
  10. EXECUTEISAPI - Runs the ISAPI script but does not need a server to run. Instead, it acts as a self server.

A simpler way to manage the MIME types is to use the Control Center Application.

In the first menu  select an extension for the MIME type (html). In the second menu select  the MIME description to be used (text/html). In the third menu select the method used for sending the file (SEND THE FILE AS IT IS). Finally, indicate the optional path to be used  in the text box under the second menu (Shown here to be empty).

To add a new extension click on the "Add a new extension" button.  To add a new MIME type click on the "Add a new MIME type" button.

Click OK to save your modifications..
