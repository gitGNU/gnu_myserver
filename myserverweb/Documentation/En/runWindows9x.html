STARTING MYSERVER

Microsoft Windows

You can run MyServer under Windows 98, NT and XP in two ways:



1. You can run the myserver.exe command from the start menu

a. Open Start Menu

b. Click on Run

c. Type in C:\MyServer\myserver.exe(change C:\MyServer\ with your MyServer installation path)


Running the program from the Start Menu


2. If you are running MyServer on the Windows NT family (NT, 2000, XP, .NET) you can also access the program through the Control Center and run it as a Service.  For directions on how to do this, please consult the  RUN LIKE A SERVICE page.
