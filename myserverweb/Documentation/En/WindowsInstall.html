Installation Guide for Windows

To install MyServer on Windows:

1.  Download the compressed file WIN32EXEC-0.4.1.zip

2.  Unzip the file (Uncompress on Windows XP).

3.  Run the setup program.

Unless otherwise indicated during the setup process, MyServer will create a new program file in the program directory.  You can then access this program through the Start Up menu.

The following Screen Shots will assist you in installing the program in Windows:


    
    Accept the Licensing Agreement


   
         Choose the components you wish to install (Expert Users)
           Most users will accept the default screen


    
         The program automatically installs itself in the Program files directory
 

     
         Install a shortcut in the Start Menu


After a few seconds, you will be able to run MyServer from the Start Menu.
