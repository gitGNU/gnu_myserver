WinCGI

WinCGI is a method for writing CGI scripts in Visual Basic. It's not a "web standard", as it will work only on servers running in Windows environment.

All you'd need to do to run your WinCGI script, is to use the script as the 'action' of your web form, like this:

<FORM METHOD=POST ACTION=www.myfiles.com/cgi-bin/mywincgi.exe>

With this, MyServer will decode and send the data of your form to the WinCGI script.

Like in CGI, you can place all your scripts in a cgi-bin directory in the web directory in MyServer as seen below:
