ERROR PAGES

You can modify the html pages that are sent when an error occurs.

The default files are contained in the system folder used by the host and can be replaced with your own files.

The pages must not contain any references to other documents in the system folder.

 Click here for informations on how configure the virtual hosts.



  The famous 404 error happens when a requested file cannot be found on the server.

 For  more information on the error pages consult the RFC2068 document found at  www.w3c.org.
