HELP US

If you are a developer and are interested in this project you can help us.
You can view the TODO list on the project site www.myserverweb.sourceforge.net.
If you are interested please contact the project administrator.


If you are not a developer, there are other ways to help:

      translating myServer to your language,
improving the documentation quality,
testing it and reporting bugs,
writing sample scripts in Perl, PHP and any CGI

To report new bugs please use the SourceForge bugs tracking system.

If you are interested in donating  to this project go to the project site  and click on the Donation button.

If you have any questions or comments please post a message in the forum.
