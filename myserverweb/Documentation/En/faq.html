FAQ

What is myServer?

myServer is a free, powerful web server.  It is designed to be used on a home PC
 and is extremely simple to install and use

What is a web server?

A web server is an application that allows clients view web pages and applications.

How can clients view my documents?

Simply with a browser. When myServer is running and you are online you can be accessed simple accessing this url from a browser: http:\\a.b.c.d:e where a.b.c.d is your IP address and e is the port used for the HTTP server.

Why use myServer?

Because myServer is a simple webserver that anyone can use. You can transform your home PC in a server that can share information with the world. It is different from other web server because it is freely available and it does not require advanced knowledge of web servers to use.

Why myServer is free?

myServer is free because it uses the GNU General Public License, please read the license page for more information.

How can I see my web when I am offline?

You don't need to be online to view the  web pages on your local machine. While myServer is running, open the browser and connect to this url: http:\\127.0.0.1 if you use a port number different from port 80 you will need to specify the port number also. For example, if you use port 10 you must connect to http:\\127.0.0.1:10.
Configuring the virtual hosts allows you to change port numbers.
