CVS

myServer is in continuos development so, if you want to have the latest files you MUST use the cvs system.  The files available in the CVS repository often have more functionality than the files that are available for download.  However, the files are released to the general public after testing of the new features implemented, so released files are more stable than the files available in the CVS repository.

If you don't have the CVS client visit:www.cvshome.org.

 

CVS HELP FOR ANONYMOUS LOGIN

To get the latest source available use the CVS client from the command line:

1. Login:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb login

2.  The first time that you copy the CVS contents type:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb co myserverweb

3. If you want to update your files type:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb update myserverweb
