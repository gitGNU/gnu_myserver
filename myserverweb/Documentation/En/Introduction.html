MyServer is a free and easily configurable web server available for both the
Windows and Linux Operating Systems.

It is licensed under the GNU Public License (GPL). Please see the license page for additional information.


MyServer's principal goal is to create a free, and  powerful application to allow anyone to transform his or her  home PC into a web server and share web files easily.  As it is a multhithread application that supports multiprocessor machines, it can also  be used professionally.

Visit the myServer site http://myserverweb.sourceforge.net for more information and to get the latest news about this project.

You can use the site forum to post any questions or comments you may have about the project.
