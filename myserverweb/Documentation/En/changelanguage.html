CHANGING THE DEFAULT LANGUAGE

You can change the language used by myServer by modifing the file myserver.xml.

  Find the line:

<LANGUAGE>english.xml</LANGUAGE>

and replace english.xml the name of the language that you want to use.

You can view the languages that are available in the folder languages.

If you want to traslate the myServer documentation into another language please do so by using the english.xml file.  Place the translated documentation into a new file with the name of your language indicated. Please contact the project administrator when you finish your translation so the new language can be included in the next release of myServer.
