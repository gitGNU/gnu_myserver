CVS


MyServer is in continuos development so, if you want to have the latest files you should use the cvs system. The files available in the CVS repository often have more functionality than the files that are available for download. However, the files are released to the general public after testing of the new features implemented, released files are more stable than the files available in the CVS repository, we suggest you to use only official releases in production environments.

Download CVS client

If you don't have the CVS client visit: www.cvshome.org.

Anonymous login

To get the latest source available use the CVS client from the command line:

1. Login:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb login

2.  The first time that you copy the CVS contents type:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb copy myserverweb

3. If you want to update your files type:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb update myserverweb

There are CVS clients available to dowload for the Windows operating system. One such client is Tortoise CVS.
