ISAPI

ISAPI (Internet Server Application Program Interface) is a protocol used by Windows based computers to run an application directly from the Web page instead of having the server process the application as a CGI script.  In order to use ISAPI, you must create a DLL (dynamic link library) that is run as part of the HTTP application.

The DLL is loaded in to the calling computer's memory when the web page is opened and remains there until the user closes the web page.  ISAPI is a faster way to process applications as it does not require any additional action by the Web Server.

Warning: If an ISAPI application crashes it may cause the calling application to fail also.


For more information, you can read Microsoft's ISAPI Overview.
