HTML FILES


HTML is used to write static web pages, and is used in conjunction with other languages to create dynamic pages.  Anyone can create HTML (Hypertext Markup Language) files that can be viewed through MyServer.


The easiest way to create HTML files is to use a word processor such as Open Office or Microsoft Word and save the file with the .html extension.  You can also create HTML files by writing the code in a text editor such as notepad or emacs.


The default place to save HTML files is in the web folder. 





The default "web" folder is selected.





You can define another folder by configuring the virtual hosts.
