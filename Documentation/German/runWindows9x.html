STARTEN VON MYSERVER

Microsoft Windows

Man kann MyServer unter Windows 98, NT und XP auf zwei Arten ausführen:



1. Man kann myserver.exe aus dem StartmenÜ ausführen

a. Öffnen Sie das Startmenü

b. Klicken Sie auf Ausführen

c. Tippen Sie C:\MyServer\myserver.exe (ändern Sie C:\MyServer\ in Ihr MyServer Installationsverzeichnis) ein


Ausführen des Programms vom Startmenü


2. Wenn Sie MyServer auf der Windows NT family (NT, 2000, XP, .NET) laufen lassen, können Sie auf das Programm durch das Control Center zugreifen und es als Service ausführen. Um Hinweise dazu zu bekommen, lesen Sie bitte die MYSERVER ALS SERVICE AUSFÜHREN Dokumentation.
