IHRE WEBSITE ANSEHEN


Nachdem Sie die Webseiten im richtigen Verzeichnis abgelegt haben, werden Sie sicher Sie von dem Computer aus, auf dem Sie liegen und vom Internet aus betrachten.

Um die Dateien auf Ihrem lokalen Computer zu betrachten, tippen Sie:

http://localhost

in der Adressleiste Ihres Browsers.

Um Ihre Dateien über das Internet zu betrachten, können Sie die IP Adresse Ihres Computers in jeden Browser eintippen. Wenn Ihre IP Adresse zum Beispiel 36.65.34.32 lautet, finden Sie Ihre Dateien auf:

http://36.65.34.32.

Ich hasse meine IP Adresse

Wenn Sie http://william.homeip.net anstatt Ihrer IP Adresse bevorzugen, müssen Sie einen DNS Namen anmelden.


Ein DNS Name ist ein Netzwerkmechanismus, der den IP adressen lesbare DNS Namen, wie www.google.com zuweist. Sie können einen kostenlosen DNS Service, wie DynDNS.org benutzen. Dieser Service ist für Computer, die ständig online sind (ADSL, Kabel,...), Verbindungen die zu jeder Tageszeit erreichbar sind.


Falls Sie keine statische IP Adresse haben, ändert sich Ihre IP Adresse jedes Mal, wenn Sie sich zum Internet verbinden. Aber mit einem DNS Namen von DynDNS.org können Sie sowohl dynamische, als auch statische IP Adressen benutzten.


Um sicher zu gehen, dass Ihr neuer DNS Name jedes Mal auf Ihren Computer verweist, wenn Sie sich zum Internet verbinden, müssen Sie einen dynamischen DNS Client installieren. Einen freien DNS Client bekommen Sie unter: http://www.sitedevelopers.com.


Wenn Ihnen ein Name, wie http://myName.homeip.net nicht gefällt und Sie einen Namen wie http://www.myName.net bevorzugen, müssen Sie eine Domain bei einem Provider registrieren, durch den weitere kosten entstehen.
