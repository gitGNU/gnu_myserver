FastCGI

FastCGI ist ein Erweiterung zu CGI, die nicht von der Sprache, die benutzt wird abhängig ist, bietet große Leistung für Anwendungen, indem man nicht Server spezifische APIs benutzt.
In FastCGI, hat man nicht mehr einen seperaten Prozess für jede Skriptanfrage.

Sie können FastCGI sehr einfach in MyServer benutzten, indem Sie die MIME Type Einstellungen für Ihre Skriptsprache auf führe als FastCGI aus setzen (benutzen Sie RUNFASTCGI).

Lesen Sie den PHP Abschnitt, um zu erfahren, wie man PHP+FastCGI benutzt.

Wie auch bei CGI, können Sie ein cgi-bin Verzeichnis im web Verzeichnis von MyServer plazieren, wie unten gezeigt wird, und all Ihre Anwendungen dort ablegen:



Das cgi-bin Verzeichnis ist direkt unter dem markierten web Verzeichnis.



Wie auch immer, CGI scripts können überall plaziert werden, weil der Webserver alles weitere durch seine Erweiterungen erfährt.
