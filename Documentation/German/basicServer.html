SERVER GRUNDEINSTELLUNGEN

In der Datei myserver.xml kann man die Startseite für Ihre mit MyServer gehosteten Webseiten.

Folgende Zeilen sind bereits in der Konfigurationsdatei enthalten:

    <DEFAULT_FILENAME0>default.html</DEFAULT_FILENAME0>
    <DEFAULT_FILENAME1>default.php</DEFAULT_FILENAME1>
    <DEFAULT_FILENAME2>default.php3</DEFAULT_FILENAME2>
    <DEFAULT_FILENAME3>index.html</DEFAULT_FILENAME3>
    <DEFAULT_FILENAME4>index.php</DEFAULT_FILENAME4>
    <DEFAULT_FILENAME5>index.php3</DEFAULT_FILENAME5>
    <DEFAULT_FILENAME6>default.htm</DEFAULT_FILENAME6>


Falls Sie einen Virtuellen Host http://mypc.net besitzen, wird die angezeigte Seite, wenn Sie auf diese Adresse in ihrem Browser zugreifen, eine von den oben angegebenen Dateien sein.
Sie können mehr Zeilen hinzufügen oder eine dieser Zeilen ändern, wenn Sie z.B. einen anderen Dateinamen brauchen. Der selbe Mechnismus wird für jedes
Verzeichnis und nicht nur für das Wurzelverzeichnis benutzt.


Im Abschnitt Fehler haben Sie gelernt, wie man eigene Fehlerseiten erstellt und benutzt. Sie können aber auch die standard Fehlerseiten deaktivieren,
indem Sie folgende Zeile ändern:
<USE_ERRORS_FILES>YES</USE_ERRORS_FILES>
Wenn Sie den Wert auf NO setzten, wie in der nächsten Zeile
<USE_ERRORS_FILES>NO</USE_ERRORS_FILES>
erhalten Sie TEXT Nachrichten anstatt von HTML Seiten für auftretende Fehler.
Sie können das Aussehen der Ordnerinhalts-Seite ändern (, die angezeigt wird, wenn Sie keine Standard-Seite haben, diese wird im Abschnitt Ordnerinhalte durchsehen beschrieben).
Folgende Zeile, in der Datei in myserver.xml, verweist auf eine CSS Datei, die benutzt wird um das Aussehen dieser Seite zu konfigurieren:
<BROWSEFOLDER_CSS>system/folderstyle.css</BROWSEFOLDER_CSS>

CGI Skripte können die Server Administrator E-mail Adresse auslesen und sie dem Benutzer anzeigen. Deshalb ist es von großer Bedeutung diese Adresse ordentlich anzugeben,
indem Sie den Wert in der Datei myserver.xml, wie in folgender Zeile gezeigt wird, ändern:
<SERVER_ADMIN>administrator@localhost</SERVER_ADMIN>
