EINE WEBSITE VERÖFFENTLICHEN


Standardmäßig wird MyServer mit einem single Host installiert, der von allen IPs des lokalen Computers aufrufbar. Die Dateien, die in der downloadbaren Version des Servers enthalten sind, sind nur für Demonstrationszwecke. Um Dateien mit anderen aus zu tauschen, sollten Sie die Dateien im web Verzeichnis mit Ihren eigenen Dateien ersetzen.

1. Löschen Sie die Dateien, die im web Verzeichnis sind
2. Fügen Sie Ihre eigenen Dateien hinzu.

Vergewissern Sie sich, dass eine Datei mit dem Namen "default.html" existiert, die die standard Datei ist, die zuerst den Benutzern angezeigt wird, die auf Ihren Server zugreifen..

Im markierten "web" Verzeichnis müssen Sie Ihre Änderungen vornehmen


Wenn Sie die standard Einstellungen für Verzeichnisse oder die IP Einstellungen ändern möchten, müssen Sie die virtuellen Hosts konfigurieren.
