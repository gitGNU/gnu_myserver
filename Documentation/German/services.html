Windows NT Family

Den Service im Betriebssystem registrieren


Wenn Sie die Windows NT family(NT, 2000, XP, .NET) benutzen, können Sie MyServer als Betriebssystem Service ausführen. Bevor Sie den Service ausführen, müssen Sie den Service zuerst registrieren.

1. Öffnen Sie das Control Center im Startmenü

2. Klicken Sie auf Install/Remove service in der Menüleiste.

3. Klicken Sie auf register service.



Den Service registrieren

Hinweis: Falls Sie den Service zu einem späteren Zeitpunkt entfernen möchten, müssen Sie auf Removeservice klicken. Das entfernt nicht das Programm, sondern nur die Registration des Services des Betriebssystems.


Starten und Stoppen des Services

Nachdem Sie den Service registriert haben, können Sie start und stop einfach im Control Center benutzen.


1. Im Menüpunkt file, wählen Sie Run as a Service um das Programm zu starten.

2. Wenn Sie das Programm beenden möchten, klicken Sie auf Stop the service im Menüpunkt file.



Starten und Stoppen des Services

Warnung Um das Control Center zu schließen, müssen Sie Exit im File Menü auswählen. Der Webserver und das Control Centerr sind zwei verschiedene Applikationen, das Schlieﬂen des Control Centers betrifft nicht den Webserver.

Mehr Informationen über die Services finden Sie auf www.microsoft.com, indem Sie nach Services suchen.
