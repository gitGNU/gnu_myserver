HILFT UNS

Wenn Sie ein Entwickler sind und an diesem Projekt interessiert sind, können Sie uns helfen.
Sie können sich die TODO Liste auf www.myserverproject.net ansehen.
Falls Sie interessiert sind, kontaktieren Sie den Projekt Administrator.


Wenn Sie kein Entwickler sind, gibt es immer noch viele andere Wege, um uns zu helfen:

MyServer in andere Sprachen übersetzen,
Die Qualität der Dokumentation verbessern,
Testen und Fehler suchen,
Beispielskripte in Perl, PHP oder CGI schreiben

Um neue Fehler zu melden, benutzen Sie das SourceForge bug tracking System.

Wenn Sie fü dieses Projekt spenden möchten, besuchen Sie die Projektseite  und clicken Sie auf den Donation button (Spendenknopf).

Wenn Sie Fragen oder Anregungen haben, benutzen Sie das Forum
