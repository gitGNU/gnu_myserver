ORDNERINHALTE DURCHSEHEN

Wenn ein Client ein Verzeichnis vom Server anfordert und der Server die standard Datei nicht finden kann, sendet der Server eine Übersicht mit dem Verzeichnisinhalt and den Client. Dies ermöglicht dem Client den Ordner durch zu sehen und auf jede Datei in diesem Verzeichnis zu zu greifen.

Übersicht des Verzeichnisinhalts


Wenn Sie ein Cascading Style Sheet (CSS) für die Verzeichnis Indexseite benutzen möchten, können Sie die Datei myserver.xml editieren.

Finden Sie diese Zeile:
<BROWSEFOLDER_CSS>system/folderstyle.css</BROWSEFOLDER_CSS>

und ersetzen Sie system/folderstyle.css mit dem Namen Ihrer .css Datei.

Die Datei system/folderstyle.css sieht standardmäßig so aus:

body { font-family: Arial; color: #000000;background-color:#FFFFFF }
