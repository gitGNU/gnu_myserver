COME USARE PERL CON MYSERVER



Perl è un linguaggio di programmazione molto usato per la programmazione di CGI. Perl permette di accedere ai databases nello stesso modo in cui permette di usare file html e xml.

Per usare Perl bisogna disporre di un interprete Perl.

GNU/Linux normalmente dispone già di un interprete Perl. Per vedere in quale directory è installato scrivi nella console:

type perl

Normalmente è installato nella directory:    /usr/bin/perl.

Per usarlo sotto windows devi prima installarlo.  Lo puoi trovare su: Active State Tool Company.

Maggiori informazioni ed esempi possono essere trovati sul sito Perl( www.perl.org) o utilizzando la documentazione distribuita con l'interprete.  Sotto Windows la documentazione è disponibile attraverso pagine HTML mentre sotto linux attraverso pagine di man.
