USARE PHP CON MYSERVER



PHP (Hypertext Preprocessor) è il più usato linguaggio di scripting server-side al mondo. Un linguaggio server-side è usato per produrre contenuti dinamici. Tu puoi usare PHP con MyServer sia come normale processo CGI che come server FastCGI. Usando PHP come CGI il serevr creerà un nuovo processo per ogni pagina richiesta mentre usandolo come FastCGI il server non necessità di un nuovo processo e si hanno migliori prestazioni. Visita il sito href="http://www.fastcgi.com/" target="_blank">FastCGI per maggiori informazioni.

Per usare PHP scaricalo da: www.php.net.

Con le versioni più aggiornate di PHP hai bisogno di modificare il file php.ini per configurare cgi.force_redirect a null. Trova la riga:

;cgi.force_redirect= 0

e rimuovi il primo punto e virgola.
In modo da essere:

cgi.force_redirect = 0

Server FastCGI remoto

MyServer può usare un FastCGI server remoto.  Il seguente tipo MIME usa una connessione remota per accedere al server FastCGI:

<MIMETYPE>
<EXT>php</EXT>
<MIME>text/html</MIME>
<CMD>RUNFASTCGI</CMD>
;


<MANAGER>@host:port</MANAGER>
</MIMETYPE>


Un server remoto deve iniziare con @.

Se stai configurando un tipo MIME attraverso la GUI aggiungi il manager nella forma:

@HOST:PORT
