CVS

MyServer è in continuo sviluppo, quindi se vuoi avere gli ultimi files devi usare il sistema CVS.  Generalmente i files disponibili attraverso il CVS implementano più funzionalità rispetto a quelli disponibili per il download. I files disponibili per il download vantano di una serie di test quindi sono più stabili rispetto a quelli sul CVS. Usa i files disponibili per il download se necessiti di un ambiente stabile.

DOWNLOAD CVS CLIENT

Se non hai un client CVS visita: www.cvshome.org.

 

LOGIN ANONIMO

Al prompt dei comandi digita per scaricare i files attraverso il CVS:

1. Login:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb login

2.  Se è la prima volta che usi il CVS per prendere i files di MyServer:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb copy myserverweb

3. Se invece vuoi aggiornare i files che già hai:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb update myserverweb



Ci sono client per windows che dispongono di una GUI. Uno di questi è Tortoise CVS.
