CGI

CGI (Common Gateway Interface) è lo standard in cui un web server manda informazioni ad una applicazione esterna e ritorna il risultato al client.  Le CGI sono generalmente scritte in linguaggi come: PHP o Perl. Il nome dell'applicazione CGI che il browser richede è scritta nel codice HTML. 

Per esempio, se stai creando il form, l'URL dello script sarà:

<FORM METHOD=POST ACTION=www.myfiles.com/cgi-bin/myprog.pl>

Per mantenere tutti gli scripts in una sola cartella puoi sostituire la cartella cgi-bin nella cartella web come visualizzato sotto:



La cartella cgi-bin è direttamente sotto quella web.



.
In ogni modo le CGI possono essere messe in qualsiasi parte nel sito.

Gli scripts CGI richiedono un nuovo processo per essere eseguiti. Una valida alternativa è quella di usare il protocollo ISAPI
