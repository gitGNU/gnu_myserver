ESEGUI COME UN SERVIZIO

Windows NT

Registra il servizio sul sistema operativo


Se stai usando un OS della famiglia Windows NT(NT, 2000, XP, .NET), tu puoi usare MyServer come un servizio del sistema operativo. Prima di tutto devi registrare il servizio.

1. Apri la Control Center Application

2. Install/Remove service.

3. Register service.



Registrando il servizio

Note: Se vuoi rimuovere il servizio in seguito puoi farlo sempre attraverso la Control Center Application usando remove service. Questo non rimuoverà il programma ma solo il servizio del sistema operativo.

Esegui e ferma il servizio

Una volta registrato il servizio lo pui facilmente eseguire e fermare dalla Control Center GUI.


1. Dal file menu, seleziona Run as a Service per eseguire il programma.

2. Quando non vuoi più il programma in esecuzione, clicca Stop the service sotto il menu file.



Eseguendo e stoppando il servizio.

Attenzione: Per chiudere la Control Center Application usa Exit. Il web server e la Control Center GUI sono due distinti programmi, chiudendo uno non si chiude l'altro.

Per maggiorin informazioni sui servizi di windows visita: www.microsoft.com
