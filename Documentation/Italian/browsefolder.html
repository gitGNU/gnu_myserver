INDICIZZA LA CARTELLA

Quando un client richiede un file che sul file system locale è una cartella il web server indicizza il contenuto della stessa e lo invia al client attraverso l'HTTP. Questo permette di vedere tutti i files all'interno di una directory e di accedere ad ognuno di essi.

Esempio di output.


Se vuoi usare un file CSS per definire lo stile della pagina modifica il file: myserver.xml.

Trova la riga:
<BROWSEFOLDER_CSS>system/folderstyle.css</BROWSEFOLDER_CSS>

e cambiala system/folderstyle.css con il nome del tuo file .css.

Il file system/folderstyle.css ha la sintassi:

body { font-family: Arial; color: #000000;background-color:#FFFFFF }
