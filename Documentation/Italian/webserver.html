WEB SERVER

Cosa è un web server?

Un web server è una applicazione per condividere i tuoi files attraverso una rete locale o attraverso una WAN come internet.

Per connetterti ad un web server tu hai bisogno solo di un browser.

Con l'evoluzione di internet il modello client-server è il modo più semplice per condividere informazioni facilmente.

Come sono i files condivisi?

Ad un livello base il server condivide pagine HTML e immagini. Questi documenti sono statici e sono i documenti che maggiormente vedi quando ti colleghi ad un sito internet.  I files sono inviati al client attraverso l'Hypertext Transfer Protocol (HTTP).

Se sono richiesti documenti dinamici - per eseguire per esempio un sito di e-commerce, sono richiesti differenti protocolli da usare sul web server.I più comuni sono: Common Gateway Interface (CGI), ISAPI, FastCGI
L'HTTP non è un protocollo sicuro perchè i dati non sono criptati, per avere una connessione sicura usa il protocollo HTTPS. Guarda il protocollo HTTPS qui.
