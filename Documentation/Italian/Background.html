MYSERVER PROJECT

MyServer è un web server gratuito e facile da configurare disponibile sia per Windows che per GNU/Linux


L'obiettivo fondamentale di Myserver è di creare un programma gratuito, e  potente per permettere a chiunque di trasformare il proprio PC domestico in un web server per condividere più facilmente i propri files. MyServer è una applicazione multithreads perciò può essere apprezzata anche per utilizzi professionali.

Visitate il nostro sito: http://www.myserverproject.net per avere più informazioni e scaricare la versione più aggiornata del server.

Il forum permette di inviare messaggi agli altri utenti ed avere un supporto tecnico.
