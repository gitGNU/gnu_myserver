MYSERVER PROJECT

MyServer è web server gratuito e facile da configurare disponibile sia per Windows che per GNU/Linux


L'obiettivo fondamentale di Myserver è di creare un programma gratuito, e  potente per permettere a chiunque di trasformare il suo PC domestico in un web server per condividere più facilmente i propri files.  MyServer è una applicazione multithread perciò può essere apprezzata anche per utilizzi professionali.

Visita il nostro sito: http://www.myserverproject.net per avere più informazioni e scaricare la versione più aggiornata del server.

Puoi usare il forum per postare messaggi agli altri utenti o per avere un supporto tecnico.
