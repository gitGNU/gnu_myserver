HTML FILES


HTML è per scrivere pagine statiche, e in unione con altri linguaggi per pagine dinamiche. Ognuno può creare HTML(HyperText Markup Language) files usati attraverso MyServer.


Il modo più efficiente di scrivere pagine HTML è attraverso un word processor come: Open Office or Microsoft Word e salvare il file con estensione .html Tu puoi anche creare documenti HTML usando un editor di testi come notepad o emacs.


I documenti da usare con il web server vanno salvati nelal cartella web





La cartella di default "web" selezionata





Si possono usare altre cartelle usando il file virtual hosts.
