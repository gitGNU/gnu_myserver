INICIANDO O MYSERVER

Microsoft Windows

Você pode rodar o MyServer de duas maneiras, como vamos comentar abaixo:



1. Você pode rodar o myserver.exe na linha de comando do Menu Iniciar
(Isso rda no Windows 98, XP, ou qualquer um dos Windows baseados no NT)

a. Abra o Menu Iniciar

b. Clique em Executar

c. Digite C:\MyServer\myserver.exe (troque o C:\MyServer\ pelo diretório onde você instalou o MyServer)


Executando o programa do Menu Iniciar


2. Se você estiver rodando o MyServer na família Windows NT (NT, 2000, XP, .NET), você pode também acessar o programa pelo Painel de Controle como um Serviço.  Para instruções sobre como fazer isso, por favor consulte a página  EXECUTAR COMO UM SERVIÇO.
