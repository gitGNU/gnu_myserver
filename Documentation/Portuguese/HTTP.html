HTTP

HTTP (Hypertext Transfer Protocol ou Protocolo de Transferência de Hipertexto) é o conjunto de regras que governam a transferência de arquivos pela Internet. Esses arquivos podem ser textos, gráficos, sons ou vídeos. O HTTP usa uma interface cliente-servidor, onde os clientes requisitam um arquivo ou aplicação e o servidor processa a requisição e retorna o arquivo certo, ou a aplicação pedida.

A maioria dos arquivos acessados estão escritos na linguagem HTML e podem conter aplicações CGI ou ISAPI.

A versão atual do HTTP é a HTTP 1.1. Mais informações podem ser encontradas no Consórcio WWW.
