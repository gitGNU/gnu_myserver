PUBLICANDO UM WEB SITE


Por default, o myServer é instalado com apenas um host acessível de todos os IPs da máquina local. Os arquivos incluídos na versão que você baixou do servidor são apenas para demostração. Para compartilhar seus arquivos com as outras pessoas, você deveria substituir os arquivos na pasta web com seus arquivos.

1. Apague os arquivos que estão atualmente no diretório web
2. Coloque os seus arquivos

Certifique-se de incluir um arquivo chamado "default.html" já que esse é o primeiro arquivo que as pessoas vão ver quando acessarem o servidor.

A pata "web" está marcada acima. É onde você faria suas alterações.


Se você gostaria de mudar as configurações da pasta default, ou configurações de IP, você tem que configurar os virtual hosts.
