ISAPI

O ISAPI (Internet Server Application Program Interface) é um protocolo usado por computadores baseados no Windows para rodar uma aplicação dinâmica carregando um módulo externo no espaço de endereçamento do processo do webserver. Para usar o ISAPI, você tem que criar uma DLL (biblioteca de vínculos dinâmicos - dynamic link library) que exporte alguns símbolos chamados pelo webserver.

Atenção: Se uma aplicação ISAPI travar, pode causar uma falha na aplicação também, de modo que pode ser uma causa de travar o MyServer.


Para mais informações, você pode ler um Overview do ISAPI da Microsoft.
