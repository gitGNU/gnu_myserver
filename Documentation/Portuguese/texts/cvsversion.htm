CVS


O MyServer está em desenvolvimento contínuo, então se você quiser os arquivos mais atuais, você deveria usar o sistema CVS. Os arquivos disponíveis no repositório CVS freqüentemente têm mais funcionalidades que os arquivos para download. Contudo, os arquivos são lançados ao público em geral após os testes das novas caracterísitcas implementadas, então os arquivos das versões lançadas são mais estáveis que os disponíveis no repositório do CVS.

Baixe o Cliente CVS

Se você não tem o cliente do CVS, visite:  www.cvshome.org.

Login Anônimo

Para obter o último fonte disponível, use o cliente CVS da linha de comando:

1. Login:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb login

2.  A primeira vez que você copiar o conteúdo do CVS:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb copy myserverweb

3. Se você quiser atualizar seus arquivos, digite:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb update myserverweb

 

Há também clientes CVS disponíveis para download para o sistema operacional Windows. Um clientes desses é o Tortoise CVS
