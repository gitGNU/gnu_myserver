NAVEGAÇÃO DE PASTAS

Quando um cliente requisita uma pasta (diretório, subdiretório) do servidor, e o servidor não conseguir encontrar um arquivo default para enviar, ele indexa a pasta e envia uma lista dos arquivos para o cliente. Isso permite ao cliente navegar pela pasta e acessar qualquer arquivo dentro dela.



Pasta indexada para Navegação

Se você quiser definir um estilo CSS para a página de índice de pastas, você pode modificar o arquivo myserver.xml.

Localize a linha:
<BROWSEFOLDER_CSS>system/folderstyle.css</BROWSEFOLDER_CSS>
e substitua system/folderstyle.css com o nome de seu arquivo .css.

O arquivo system/folderstyle.css tem a forma default:

body { font-family: Arial; color: #000000;background-color:#FFFFFF }
