CONTROLE REMOTO

O MyServer permite a você gerenciar as configurações do servidor remotamente, a partir da aplicação Control Center, de qualquer lugar no mundo. Você só precisa de habilitar o Protocolo de Controle.

Para isso, você precisa adicionar um host virtual  que vai usar o Protocolo de Controle. Você pode fazer isso modificando o arquivo  virtualhosts.xml, de modo que ele se apreça com isso:

<VHOST>
<NAME>Every control connection</NAME>
<PORT>270</PORT>
<SSL_PRIVATEKEY>certificates/server.key</SSL_PRIVATEKEY>  
<SSL_CERTIFICATE>certificates/server.pem</SSL_CERTIFICATE >  
<PROTOCOL>CONTROL</PROTOCOL>
<DOCROOT></DOCROOT>
<SYSFOLDER></SYSFOLDER>
<ACCESSESLOG>logs/control.log</ACCESSESLOG>
<WARNINGLOG>logs/control.err</WARNINGLOG>
</VHOST> 

A linha <PORT>270</PORT> descreve a porta a ser usada para acessar o Controle Remoto. No exemplo acima, você poderia acessar o Controle Remoto do seu webserver pela aplicação GUI, conectando ao localhost na porta 270.

Por favor, note que os arquivos certificates/server.key e certificates/server.pem precisam ser criados antes de serem usados. Pesquise como criar chaves SSL para ter mais informações sobre isso.

Você vai precisar também modificar o myserver.xml para setar uma senha para administração, adicionando as seguintes linhas:

<CONTROL_ENABLED>YES</CONTROL_ENABLED> 
<CONTROL_ADMIN>ADMIN</CONTROL_ADMIN> 
<CONTROL_PASSWORD>PASSWORD</CONTROL_PASSWORD> 

Basta mudar os valorer de ADMIN e PASSWORD para o seu nome de usuário administrador e senha, e lembrar-se de usar  YES no parâmetro CONTROL_ENABLED (do contrário, você não vai conseguir usar o Control Center remotamente)

Também, defina um nome e uma senha para o administrador que não sejam simples como "admin" ou "password".
