MYSERVER PROJECT

O MyServer é um servidor web livre e facilmente configurável disponível para os Sistemas Operacionais
Windows e *Nix.


O principal objetivo do MyServer é criar uma aplicação livre e poderosa que permita a qualquer pessoa transformar seu PC de casa em um servidor web e compartilhar arquivos web facilmente. Como é uma aplicação multithread, que suporta máquinas multiprocessadas, também pode ser usada profissionalmente.

Por favor visite nosso site http://www.myserverproject.net para mais informações e para obter as últimas notícias sobre este projeto.

Você pode usar o fórum do site para enviar perguntas ou comentários que você possa ter sobre o projeto.
