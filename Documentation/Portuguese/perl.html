COMO USAR PERL NO MYSERVER



Perl é outra linguagem amplamente utilizada para scripts, perfeita para manipulação de text e gerenciamente de bancos de dados na web. Perl também funciona com HTML e XML bem como outras linguagens markup. Com o módulo CGI.pm do Perl, incluído na distribuição padrão, formulários HTML são manuseados facilmente.

Para usar Perl, você precisa primeiro se certificar que tem um interpretador.

GNU/Linux já vem com o perl instalado. Para descobrir em que diretório ele está no seu sistema, digite:

type perl

Geralmente ele é encontrado no diretório /usr/bin/perl

Para utilizar o Perl no Windows, você precisa primeiro instalá-lo. O interpretador e a documentação podem ser encontrados na Active State Tool Company.

Mais informações e códigos de demonstração podem ser encontrados no site do Perl (www.perl.org) ou lendo a documentação incluída com o interpretador. No Windows, a documentação está disponível nos arquivos HTML incluídos no download. No GNU/Linux a documentação está disponível nos Man Pages.
