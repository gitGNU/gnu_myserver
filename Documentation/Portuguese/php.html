COMO USAR PHP NO MYSERVER



PHP (Hypertext Preprocessor) é a linguagem de script server-side mais usada no mundo. Uma linguagem server-side é usada para criar páginas web com conteúdo dinâmico. Você pode usar PHP no MyServer como um executável CGI padrão ou como um servidor Fastcgi. Usando PHP como um executável CGI, MyServer criará um novo processo para cada página requisitada. Usando PHP no modo FastCGI, o MyServer não cria um novo proceso para cada request, e então, o servidor tem melhor performance. Por favor, veja mais informações sobre FastCGI no site oficial do FastCGI.

Para usar PHP, você deve fazer o download do instalador em: www.php.net.

Com as últimas versões do PHP você vai precisar modificar o arquivo php.ini para setar a opção cgi.force_redirect para nulo. Encontre a linha abaixo no arquivo de configuração do php:

;cgi.force_redirect= 0

e remova o ponto-e-vírgula do início.
A linha ficará assim:

cgi.force_redirect = 0

Servidores FastCGI Remotos

O MyServer tem a capacidade de usar um servidor FastCGI remoto. O seguinte MIME type usa uma conexão remota:

<MIMETYPE> <EXT>php</EXT> <MIME>text/html</MIME> <CMD>RUNFASTCGI</CMD>

<MANAGER>@host:port</MANAGER> </MIMETYPE>


A definição para um gerente remoto deve começar com @.

Se você estiver configurando os MIME types pela aplicação Centro de Controle, você pode adicionar um servidor remoto adicionando a linha abaixo no formulário

@HOST:PORT

no campo do caminho do FastCGI.
