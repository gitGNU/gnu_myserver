WEB SERVER

O que é um web server?

Um web server é uma aplicação que permite que usuários compartilhem documentos na Internet ou em uma rede local.

Para conectar-se a um webserver você só precisa de um browser.


Muitos clientes conectados a um servidor.

Essa imagem mostra como vários clientes podem estar conectados com seu servidor ao mesmo tempo. Através da rede, o pedido chega ao seu servidor. O servidor prepara uma resposta e a envia ao cliente. Com a evolução da Internet, o modelo cliente-servidor é o modo mais simples de compartilhar informação facilmente, permitindo a cada usuário decidir o que quer compartilhar com outros usuários.

Como os arquivos são compartilhados?

No nível básico, o servidor compartilha documentos HTML e imagens. Estes documentos e imagens são muito simples e estáticos e são os arquivos básicos que se vê quando você acessa um website. Esses arquivos são transferidos por um protocolo chamado Protocolo de Transferência de Hipertexto (HTTP).

Se for necessário conteúdo dinâmico, por exemplo - para rodar um site de e-commerce, há protocolos diferentes necessários. Alguns dos protocolos são o Common Gateway Interface (CGI), o ISAPI, o FastCGI, o Protocolo de Transferência de Hipertexto Seguro (HTTPS).

Se arquivos precisam ser trasferidos, há outro protocolo. O Protocolo de Transferência de Arquivos (FTP) permite a fácil transferência de arquivos, tais como programas, de um computador para outro.

HTTP não é um protocolo seguro, porque os dados transmitidos não são encriptados, para se ter uma transmissão segura, você tem que usar o protocolo HTTPS. Veja o protocolo HTTPS aqui.
