ARQUIVOS HTML


O HTML é usado para escrever páginas da web, e é usada em conjunto com outras linguagens para criar páginas dinâmicas. Qualquer um pode criar arquivos HTML (Hypertext Markup Language) que podem ser vistos por intermédio do MyServer.

O modo mais fácil de se criar arquivos HTML é usando um processador de textos como o Open Office ou o Microsoft Word e salvar o arquivo com extensão .html. Você pode também criar arquivos HTML escrevendo o código em um editor de textos como o notepad ou o emacs.


O lugar padrão para salvar arquivos HTML é na pasta web.

A pasta default "web" é a selecionada.




Você pode definir uma outra pasta, configurando os hosts virtuais.
