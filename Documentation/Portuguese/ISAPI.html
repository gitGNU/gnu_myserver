ISAPI

ISAPI (Internet Server Application Program Interface) é um protocolo usado por computadores baseados no Windows para rodar uma aplicação diretamente da página web ao invés de ter que o servidor processe a aplicação como um script CGI. Para usar o ISAPI, você tem que criar uma DLL (biblioteca de vínculos dinâmicos - dynamic link library) que é executada como parte da aplicação HTTP.

A DLL é carregada na memória do computador que está chamando o arquivo, quando a página é aberta, e continua lá até que o usuário feche a página web. ISAPI é um modo mais rápido de processar aplicações já que ele não requer qualquer ação adicional pelo servidor web.

Atenção: Se uma aplicação ISAPI travar, pode causar uma falha na aplicação também, de modo que pode ser uma causa de travar o MyServer.


Para mais informações, você pode ler um Overview do ISAPI da Microsoft.
