ISAPI

ISAPI (Internet Server Application Program Interface) is a protocol used by Windows based computers to run an application directly from the Web page instead of having the server process the application as a CGI script. In order to use ISAPI, you must create a DLL (dynamic link library) that exports some symbols called by the web server.

The DLL is loaded in to the calling computer's memory when the web page is opened and remains there until the user closes the web page. ISAPI is a faster way to process applications than standard CGI.

Warning: If an ISAPI application crashes it may cause the calling application to fail also so it can be a MyServer crash cause.


For more information, you can read Microsoft's ISAPI Overview.
