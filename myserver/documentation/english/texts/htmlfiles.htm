FICHIERS HTML


HTML est utilisé pour écrire des pages web statiques, et est utilisé avec d'autres langages pour créer des pages dynamiques. N'importe qui peut créer des fichiers HTML (Hypertext Markup Language) lisibles avec MyServer.


La façon la plus rapide de créer des fichiers HTML est d'utiliser un éditeur de texte comme Open Office ou Microsoft Word et de sauvegarder la page avec l'extension .html/ Vous pouvez aussi créer vos fichiers HTML en écrivant le code dans un éditeur de text comme le Bloc Notes ou GNU Emacs.


L'emplacement de sauvegarder par défaut des ficheirs HTML est le répertoire web.


Le répertoire par défaut "web" est sélectionné.



Vous pouvez avoir d'autres répertoires en configuration les hôtes virtuels.
