BENUTZUNG VON PERL MIT MYSERVER



Perl ist eine weitere sehr weitverbreitete Skriptsprache, die sich hervorragend zur Textverarbeitung und Datenbank Management eignet. Perl arbeitet sowohl mit HTML und XML, als auch mit anderen Markup Sprachen. Mit Perl's CGI.pm Modul, das bei der standard Distribution enthalten ist, können HTML Formularen sehr einfach verarbeitet werden.

Um Perl benutzen zu können, müssen Sie sich vergewissern, dass Sie einen Perl Interpreter haben.

Linux hat Perl schon vorinstalliert. Um herauszufinden, wo sich Perl befindet:

type perl

Es befindet sich meistens in:    /usr/bin/perl

Um Perl auf Windows benutzen zu können, müssen Sie es zuerst installieren. Der Interpreter und die Dokumentation gibt es auf der Webseite der Active State Tool Company.

Mehr Informationen und Codebeispiele findet man auf der Perl Website (www.perl.org), oder indem man die Dokumentation liest, die mit dem Interpreter geliefert wird. Unter Windows wird die Dokumentation im Installationspackage als HTML Datei mitgeliefert. Unter Linux steht die Dokumentation in den Manpages zur Verfügung.
