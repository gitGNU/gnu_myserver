CVS

MyServer ist in ständiger Entwicklung. Falls Sie die aktuellsten Quelltexte haben möchten, MÜSSEN Sie das CVS System benutzen. Die Dateien, die per CVS verfügbar sind, haben meistens eine verbesserte Funktionalität, als die Dateien, die zur Verfügung stehen.  Wie auch immer, diese Dateien werden der Öffentlichkeit nach Tests und nach dem Hinzufügegen von neuen Features veröffentlicht. Deshalb sind veröffentliche Dateien sicherer und stabiler, als die Dateien, die per CVS erhältlich sind.

DOWNLOAD DES CVS CLIENTS

Wenn Sie keinen CVS Client haben, besuchen Sie:  www.cvshome.org.

 

CVS HILFE FÜR ANONYME LOGINS

Um die aktuellsten Quelltexte zu bekommen, benutzen Sie den CVS Client von der Kommandozeile aus:

1. Login:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb login

2. Das erste Mal, wenn Sie CVS Ihalte kopieren:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb copy myserverweb

3.Wenn Sie Ihre Dateien aktualisieren möchten, tippen Sie:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb update myserverweb



Es gibt auch CVS Clients für das Windows Betriebssystem.  Einer dieser Clients ist Tortoise CVS.
