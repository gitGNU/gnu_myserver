HTML DATEIEN


HTML wird benutzt, um statische Webseiten zu schreiben und wird in Verbindung mit anderen Sprachen genutzt, um dynamische Seiten zu erstellen. Jeder kann HTML (Hypertext Markup Language) Dateien erstellen, die mit MyServer betrachtet werden können.


Am einfachsten lassen sich HTML Dateien mit Textverarbeitungsprogrammen, wie Open Office oder Microsoft Word erstellen. Speichern Sie die Dateien mit der .html Erweiterung. Sie können auch HTML Dateien erstellen, indem Sie den Code in einem Texteditor wie notepad oder emacs schreiben.


Das standard Verzeichnis, um Ihe HTML Dateien abzuspeichern ist das web Verzeichnis.





The default "web" folder is selected.





Sie können ein anderes Verzeichnis konfigurieren, indem sie die virtual hosts konfigurieren.
