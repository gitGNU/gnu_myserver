ISAPI

ISAPI (Internet Server Application Program Interface) ist ein Protokoll, das von auf Windows basierten Computern genutzt wird, um Applikationen direkt von der Webseite anstatt den Server die Applikation als ein CGI Skript verarbeiten zu lassen. Um ISAPI zu benutzen, müssen Sie eine DLL (dynamic link library) erstellen, die als Teil der HTTP Applikation ausgeführt wird.

Die DLL wird in den Speicher des zugreifenden Computers geladen, wenn die Webseite geöffnet wird und verbleibt dort bis der Benutzer die Webseite verlassen hat. ISAPI ist eine schnellere Art um Applikationen zu verarbeiten, da es keine zusätzliche Verarbeitung vom Webserver braucht.

Achtung: Wenn die ISAPI Applikation abstürzt, kann es in der Aufrufenden Applikation einen Fehler verursachen und somit einen MyServer Absturz hervorrufen.


Mehr Informationen finden Sie in der ISAPI Übersicht von Microsoft.
