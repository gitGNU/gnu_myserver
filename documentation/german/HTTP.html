HTTP

HTTP (Hypertext Transfer Protocol) ist ein Satz von Regeln, der den Transfer von Dateien über das Internet regelt. Diese Dateien können Text, Grafik, Sound oder Video sein. HTTP benutzt eine einfache Server-Client Schnittstelle, an der der Client eine Datei oder Applikation anfordert und der Server die Antwort verarbeitet und die korrekte Datei oder Applikation zurückgibt.

Die meisten abgerufenen Dateien sind in der Sprache HTML geschrieben und können CGI oder ISAPI Applikationen beinhalten.

Die aktuelle Version von HTTP ist HTTP 1.1. Mehr Informationen können Sie auf der Seite des WWW Consortium finden.
