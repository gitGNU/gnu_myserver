ARCHIVOS HTML


Se usa HTML para escribir páginas Web estáticas y se usa junto con otros lenguajes para crear páginas dinámicas. Cualquiera puede crear archivo HTML (Hypertext Markup Language) que pueden ser vistos a través de MyServer.


La manera más fácil de crear archivos HTML es con procesadores de texto como Open Office o Microsoft Word y guardar el archivo con la extensión .html. También se puede crear archivos HTML al escribir el código en un editor de texto como el Bloc de Notas o GNU Emacs.


El lugar predeterminado para guardar archivos HTML es en el directorio Web.


Se selecciona el directorio "Web" predeterminado.


Puede definir otro directorio al configurar los Hosts Virtuales.
