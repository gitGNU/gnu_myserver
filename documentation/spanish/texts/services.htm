EJECUTAR COMO SERVICIO

Familia Windows NT

Registrando el Servicio en el Sistema Operativo


Si está usando la familia Windows NT (NT, 2000, XP, .NET), puede ejecutar MyService como un Servicio del Sistema Operativo. Antes de ejecutar el servicio debe registrarlo como servicio bajo el SO.

1. Abra el Centro de Control desde el Menú Inicio.

2. Haga Clic en Instalar/Quitar servicio en la barra de herramientas.

3. Haga Clic en Registrar Servicio.



Registrando el Servicio

Nota: Si quisiera quitar el servicio más tarde, podrá hacerlo al hacer clic en quitar servicio. Esto no quitará el programa, sólo quitará la registración del servicio en el SO.


Iniciar y Detener el Servicio

Una vez que registró el servicio en el SO, puede iniciar y detener el servicio fácilmente desde el Centro de Control.


1. Dentro del menú archivo, elija Ejecutar un Servicio para iniciar el programa.

2. Cuando no quiera que el programa se ejecute, haga clic en Detener el sericio dentro del menú archivo.



Iniciar y Detener el Servicio

Adevertencia Para poder cerrar el Centro de Control tenga cuidado al elegir el comando salir. El servidor Web y el Centro de Control son dos aplicaciones distintas, cerrar el Centro de Control no afectará el servidor Web.

Para más información sobre los servicios visite el sitio de Micosoft en www.microsoft.com y busque por servicios.
