HTTP

HTTP (Hypertext Transfer Protocol) es el conjunto de reglas (protocolo) que controla la transferencia de archivos a través de Internet. Estos archivos pueden ser textos, gráficos, sonido o video. HTTP usa una interfaz servidor-cliente, en donde el cliente solicita un archivo o aplicación y el servidor procesa la solicitud y devuelve la información correcta al cliente. El protocolo HTTP trabaja a un nivel 7 de la pila ISO-OSI.

El protocolo HTTP es el standard de hecho la transferencia de documentos a través de Internet.

La versión actual de HTTP es HTTP/1.1 se describe en RFC 2616. Se puede encontrar más información en WWW Consortium.
