COMO USAR PERL BAJO MYSERVER


Perl es otro lenguaje de automatización extensamente usado para la manipulación de texto y manejo de bases de datos Web. Perl también trabaja con HTML y XML como otros lenguajes de marcado. Con el módulo CGI.pm de Perl, incluido con la distribución estándar, los formularios HTML son tratados fácilmente.

Para poder usar Perl, primero debe asegurarse de que tiene un intérprete.

En sistemas parecidos a UNIX para encontrar a perl, use el comando:

escriba perl


Para poder usar Perl bajo Windows primero debe instalarlo. El intérprete y la documentación los encuentra en Active State Tool Company.

Puede encontrar más información y más ejemplos de código en el sitio de Perl (www.perl.org) o al leer la documentación incluida con el intérprete.
