CONTROL REMOTO

MyServer permite cambiar su configuración de manera remota, a través de su aplicación Centro de Control, desde cualquier parte del mundo. Sólo necesita habilitar el Protocolo de Control.

Para ello, debe agregar un host virtual que usará el Protocolo de Control. Puede hacer esto al modificar el archivo virtualhosts.xml, así que se parece a:

WARNINGLOG Advertencia, Advertencia, ADVERTENCIA, Advertencia, Advertencia, ADVERTENCIA, Advertencia, Advertencia, ADVERTENCIA, Advertencia, Advertencia, Advertencia, Advertencia, Advertencia PROTOCOL Nombre de protocolo, Protocolo, Protocolo, Protocolo, Protocolo, Protocolo, Protocolo, Protocolo server.pem Servidores, Servidores, Servidores, Servidores, Servidores server.key Servidores, Servidores, Servidores, Servidores, Servidores NAME Nombre, Nombre, Nombre, Nombre completo, Nombre, Nombre, Nombre, Nombre, Nombre, Nombre, Nombre, Nombre, Nombre, Nombre, Nombre connection Conexión, Conexión, Conexión, Conexión, Conexión, Conexión, Conexión<VHOST>
<NAME>Every control connection</NAME>
<PORT>270</PORT>
<SSL_PRIVATEKEY>certificates/server.key</SSL_PRIVATEKEY>  
<SSL_CERTIFICATE>certificates/server.pem</SSL_CERTIFICATE >  
<PROTOCOL>CONTROL</PROTOCOL>
<DOCROOT></DOCROOT>
<SYSFOLDER></SYSFOLDER>
<ACCESSESLOG>logs/control.log</ACCESSESLOG>
<WARNINGLOG>logs/control.err</WARNINGLOG>
</VHOST> 

La línea <PORT>270</PORT> describe el puerto a usar para el acceso del Control Remoto. En el ejemplo de arriba puede acceder el Control Remoto de su servidor Web al conectarse a su host local en el puerto 270

Por favor note que certificates/server.key y certificates/server.pem se deben crear para poder usarlos. Consulte como crear llaves SSL para más información sobre el tema.

También necesita modificar myserver.xml para establecer una contraseña de la administración al agregar las siguientes líneas:

<CONTROL_ENABLED>YES</CONTROL_ENABLED> 
<CONTROL_ADMIN>ADMIN</CONTROL_ADMIN> 
<CONTROL_PASSWORD>PASSWORD</CONTROL_PASSWORD> 

Sólo cambie el valor de ADMIN y PASSWORD para que se adapten a su nombre de administrador y contraseña, y recuerde usar YES en el parámetro CONTROL_ENABLED (de otra manera, no podrá usar el centro de Control Remoto.)

También define un nombre y una contraseña para el administrador, es mejor si NO usa palabras simples como admin o contraseña.
