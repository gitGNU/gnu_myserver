VER SU SITIO WEB


Una vez que sus páginas Web estén guardadas en el directorio correcto querrá verlas tanto en el equipo en la que las guardó como también a través de Internet.

Para ver sus archivos en máquina local escriba:

http://localhost

en la ventana de direcciones del explorador Web.

Para ver sus archivos en Internet, puede escribir la dirección IP de su computadora en cualquier explorador Web. Por ejemplo si su dirección IP es 36.65.34.32 sus archivos pueden ser encontrados en:

http://36.65.34.32.

Usar un nombre DNS

Si prefiere http://william.homeip.net en vez de su dirección IP debe obtener un nombre DNS.

Un nombre DNS es un mecanismo para asignar una dirección IP a un nombre mnemotécnico como www.google.com. Puede conseguir un nombre DNS en un sistema gratuito como DynDNS.org. Este servicio es para PCs siempre conectadas (ADSL,cable,...) que pueden ser accedidas en la mayor parte del día.

Si no tiene una dirección de IP estática, su dirección cambia cada vez que se conecta a internet. Pero, con un nombre DND desde DynDNS.org usted puede acceder independientemente de que su dirección IP es dinámica o estática.

Para asegurarse un nombre DNS hace referencia a su equipo cada vez que se conecta a Internet, debe instalar un cliente DNS dinámico. Puede conseguir un cliente DNS gratuito en: http://www.directupdate.net.

Si no le gusta el nombre http://myName.homeip.net pero le gustaría un nombre como http://www.myName.net necesita registrar un dominio a través de un registro.
