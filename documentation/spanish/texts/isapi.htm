ISAPI


ISAPI (Internet Server Application Program Interface) es un protocolo usado por computadoras basadas en Windows para ejecutar la carga de aplicaciones dinámicas en un módulo del Servidor Web de Espacio de direcciones del proceso. Para poder ISAPI, se debe crear un DLL (dynamic link library) que exporta algunos símbolos llamados por el servidor Web.

Advertencia: Si una aplicación ISAPI falla, puede causar que la aplicación que la ejecutó también falle y puede ser una causa para que MyServer falle.


Para más información, puede leer la Información general sobre ISAPI de Microsoft.
