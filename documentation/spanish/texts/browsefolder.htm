EXAMINAR DIRECTORIO

Cuando un cliente solicita un directorio del servidor y el servidor no encuentra un archivo por defecto a enviar, crea un índice y envía una llista de los archivos al cliente. Esto permite al cliente explorar el directorio y acceder a cualquier archivo en él.

Carpeta indexada para Navegación


Si quiere definir una hoja de estilos en cascada para la página índice del directorio, puede modificar el archivo myserver.xml.

Ubicar la línea: <BROWSEFOLDER_CSS>system/folderstyle.css</BROWSEFOLDER_CSS>

y reemplazar system/folderstyle.css con el nombre de su archivo css.

El archivo system/folderstyle.css tiene la forma por defecto:

family Familiares font Fuente, Fuente, Fuente, Fuente, Fuente, Fuente, Fuentebody { font-family: color: Color:, Color:Arial; color: background-color """Color de fondo"""#000000;background-color:#FFFFFF }
