COMO USAR PHP BAJO MYSERVER



PHP (Preprocesador de Hipertexto) es el lenguaje de automatización del lado del servidor más usado en el mundo. Un lenguaje "del lado del servidor" se usa para crear páginas Web con contenido dinámico. Puede usar PHP con MyServer con el CGI ejecutable estándar o como servidor FastCGI. Mientras use a PHP como un CGI ejecutable. MyServer creará procesos nuevos por cada página solicitada. Mientras que use a PHP en modo FastCGI, MyServer no requiere un proceso nuevo por cada solicitud y por la tanto, el servidor funciona mejor. Por favor consulte el sitio oficial FastCGI para más información sobre FastCGI.

Para poder usar PHP, primero debe descargar los archivos. Aquí hay algunos vínculos...

Usuarios Win32: Instale php en su directorio favorito: El predeterminado es C:\php

Ahora que instaló el paquete PHP. Necesita configurar MyServer para poder usar php.
Comenzaremos por agregar los tipos MIME de PHP. Esta guía agregará los tipos MIME por GUI.

1. Abra GUI de MyServer y haga clic en la pestaña MIME.

2. Debajo de la lista de extensiones listadas, haga clic en agregar. Escriba php, y pulse en ok.


3. Ahora, encuentre php en la lista de extensiones y haga clic en él. Establezca el tipo MIME en: text/html.
Entonces establezca la acción en: Ejecute como CGI, o ejecute como FASTCGI. Esta es la preferencia principal, cualquiera de las dos funcionará.
Finalmente, necesita establecer el administrador. Para sistemas Linux, ejecute el comando "which php". Ponga el resultado en el campo del administrador.
Para sistemas Win32, usted debe establecer la ubicación del archivo ejecutable aquí. Si instaló php en C:\php. Pondría C:\php\php.exe aquí.

También puede editar MIMEtypes.xml automáticamente. Puede seguir la documentación Tipos MIME. Aquí hay un ejemplo:
MANAGER Administrador, Administrador, Administrador MANAGER>C Administrar..., Administrar... CMD Comando<MIMETYPE>
<EXT>php</EXT>
<MIME>text/html</MIME>
<CMD>RUNFASTCGI</CMD>
<MANAGER>C:\php\php.exe</MANAGER>
</MIMETYPE>

4.    Ahora que MyServer está listo para ejecutar php, necesita modificar una línea en php.ini. Necesita establecer cgi.force_redirect a nulo. Encuentre la línea en el archivo de configuración php:

;cgi.force_redirect= 0

Nota: cgi.force_redirect puede ser 1. Si es así, entonces cámbielo a 0.
Luego quite el primer punto y coma. Entonces se leerá:

cgi.force_redirect = 0

¡Felicitaciones! Ha finalizado de instalar php en MyServer.
