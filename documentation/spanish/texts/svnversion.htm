Subversion


MyServer se encuentra en desarrollo así que si quiere los últimos archivos, debe usar el sistema de Subversión. Los archivos en el repositorio de Subversión a menudo tienen más funciones que los archivos que están disponibles para descargar. Sin embargo, los archivos son liberados al público después de que las nuevas funciones implementadas sean probadas, los archivos liberados son más estables que los archivos disponibles en el repositorio de desarrollo, sugerimos que use solamente las versiones oficiales en entornos de producción.

Descargar un cliente de Subversión


Si no tiene un cliente de Subversión visite: http://subversion.tigris.org/.

Inicio de sesión anónimo


Para conseguir el código más reciente use el cliente desde la línea de comando:

1.  La primera vez que copie el tipo de contenido del repositorio svn:

svn co https://myserver.svn.sourceforge.net/svnroot/myserver/trunk/myserver myserver

2. Si quiere actualizar sus archivos sólo escriba desde el directorio de myserver:

svn update

Hay clientes de Subversión disponibles para descargar para el SO Windows. Tortoise CVS Es de ese tipo de clientes.
