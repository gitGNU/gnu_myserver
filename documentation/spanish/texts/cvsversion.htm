CVS


MyServer se encuentra en desarrollo así que si quiere los últimos archivos, debería usar el sistema CVS. Los archivos en el depósito CVS a menudo tienen más funciones que los archivos que están disponibles para descargar. Sin embargo, los archivos son liberados al público después de que las nuevas funciones implementadas sean probadas, los archivos liberados son más estables que los archivos disponibles en el depósito CVS, sugerimos que use solamente las versiones oficiales en entornos de producción.

Descargue el Cliente CVS

Si no tiene el cliente CVS visite: www.cvshome.org.

Inicio de sesión anónimo

Para conseguir el código más reciente use el cliente CVS desde la línea de comando:

1. Inicio de sesión:

cvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb login

2. La primera vez que copie el tipo de contenido CVS:

copy Copiar, Copiar, Copia, Copia, Copiar, Copiar, Copiar, Copiar, Copiar, Copiar, Copiarcvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb copy myserverweb

3. Si quiere actualizar los tipos de archivos:

update Actualizar, Actualizar, Actualizar, Actualizar, Actualización, Actualizar, Actualizaciones, Actualizar, Actualizarcvs -d:pserver:anonymous@cvs.myserverweb.sourceforge.net:/cvsroot/myserverweb update myserverweb

Hay clientes CVS disponibles para descargar para el sistema operativo Windows. Tortoise CVS Es de ese tipo de clientes.
