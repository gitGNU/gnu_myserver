PUBLICAR UN SITIO WEB


MyServer se instala de manera predeterminada con un sólo host accesible desde todas las direcciones IP de la máquina local. Los archivos incluidos en la versión descargable del servidor sólo son para demostración. Para poder compartir sus archivos con otros debería reemplazar los archivos en el directorio Web con los suyos.

1. Eliminar los archivos que están actualmente en la carpeta Web
2. Agregue sus propios archivos.
Asegúrese de incluir un archivo llamado "default.html" ya que será el primero que las personas verán cuando accedan al servidor.

El directorio "Web" resaltado es dónde hará sus cambios

Si quisiera cambiar la carpeta de Configuración predeterminada o dirección IP debe configurar los hosts virtuales.
