TIPI MIME

Puoi modificare i tipi MIME usati da MyServer attraverso il file "MIMEtypes.xml" con un semplice editor di testi. Il seguente è un esempio del file:

<MIMETYPES>
-
    <MIMETYPE>
-
    <MIMETYPE>
<EXT>htm</EXT>
<MIME>text/html</MIME>
<CMD>SEND</CMD>
<MANAGER>NONE</MANAGER>
</MIMETYPE>


Ogni blocco rappresenta un tipo MIME differente.

<EXT> è l'estensione del file.

<MIME> è il tipo MIME.

<CMD> è il modo in cui usare questo tipo di files.

<MANAGER> è il manager usato per usare il tipo di file.

I seguenti valori sono validi per la sezione <CMD>

  1. SEND - Invia il file in modo statico senza nessun intervento del web server.
  2. RUNCGI - Esegue lo script CGI specificato nella path. Usato da files tipo .php e .pl che necessitano un intervento esterno.
  3. RUNFASTCGI - Esegue il server FastCGI. Anche questo può essere usato con file come .php e .pl
  4. EXECUTEFASTCGI - Anche questo esegue uno script FastCGI ma differentemente dal precedente questo viene eseguito come un server a sè stante.
  5. RUNMSCGI - Esegue uno script MSCGI direttamente implementato nel web server. Usato per scrivere CGIs molto efficienti per MyServer.
  6. EXECUTE - Il file è un file eseguibile. Viene eseguito attraverso CGI.
  7. SENDLINK - Invia il file la cui path è contenuta all'interno di questo tipo di files.
  8. EXECUTEWINCGI - Esegue come WinCGI
  9. RUNISAPI - Esegue attraverso ISAPI.
  10. EXECUTEISAPI - Esegue come un modulo ISAPI a sè stante.
Molti tipi MIME sono già presenti nel file .xml file. Se vuoi aggiungerne altri puoi farlo semplicamente aggiungendone altri alla file del file come nell'esempio.
