ESEGUI MYSERVER

Microsoft Windows

Puoi eseguire MyServer sotto windows in due modi:



1. Puoi eseguirlo dalla riga di comando:

a. Clicca sul menu start

b. Clicca su Esegui

c. Immetti C:\MyServer\myserver.exe(metti al posto di C:\MyServer\ la path alla tua installazione di MyServer)


Eseguire il programma dal menu


2. Se tu esegui MyServer sulla famiglia NT (NT, 2000, XP, .NET), tu puoi accedere al programma anche attraverso l'utility control eseguendolo come Servizio. Per maggiori informazioni, consulta la paginaRUN LIKE A SERVICE.
