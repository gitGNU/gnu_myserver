HTTP

HTTP (Hypertext Transfer Protocol) è l'insieme di regole che permette il trasferimento di dati su una connessione TCP. Questi files possono essere di ogni tipo: testo, immagini, suoni, video. La richiesta viene fatta da un client al server che cerca di soddisfarla nel miglior modo possibile.
Generalmente i siti sono scritti in HTML language e possono avere contenuti dinamici come CGI e ISAPI

La versione attuale di HTTP è HTTP 1.1.  Per maggiori informazioni: WWW Consortium.
