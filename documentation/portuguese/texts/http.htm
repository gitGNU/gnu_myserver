HTTP

O HTTP (Hypertext Transfer Protocol ou Protocolo de Transferência de Hipertexto) é o conjunto de regras que governam a transferência de arquivos pela Internet. Esses arquivos podem ser textos, gráficos, sons ou vídeos. O HTTP usa uma interface cliente-servidor, onde os clientes requisitam um arquivo ou aplicação e o servidor processa a requisição e retorna os dados certos para o cliente. O protocolo HTTP trabalha no nível 7 da pilha ISO-OSI.

O protocolo HTTP é o padrão de facto standard para transferência de documentos na Internet.

A versão atual do HTTP é a HTTP 1.1, descrita no RFC 2616. Mais informações podem ser encontradas no Consórcio WWW.
