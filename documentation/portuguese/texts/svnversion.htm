Subversion


O MyServer está em desenvolvimento contínuo, então se você quiser os arquivos mais atuais, você deveria usar o sistema Subversion. Os arquivos disponíveis no repositório Subversion freqüentemente têm mais funcionalidades que os arquivos para download. Contudo, os arquivos são lançados ao público em geral após os testes das novas caracterísitcas implementadas, então os arquivos das versões lançadas são mais estáveis que os disponíveis no repositório do Subversion.

Baixe o Cliente Subversion



Se você não tem o cliente do Subversion, visite:  http://subversion.tigris.org/.

Login Anônimo



Para obter o último fonte disponível, use o cliente Subversion da linha de comando:


1.  A primeira vez que você copiar o conteúdo do Subversion:

svn co https://myserver.svn.sourceforge.net/svnroot/myserver/trunk/myserver myserver

2. Se você quiser atualizar seus arquivos, digite:

svn update



Documentação do MyServer

Pra obter a versão mais recente disponível, use o cliente da linha de comando:


1.  A primeira vez que você copiar o conteúdo do Subversion:

svn co https://myserver.svn.sourceforge.net/svnroot/myserver/trunk/documentation documentation

2. Se você quiser atualizar seus arquivos, digite dntro do diretório "documentation":

svn update





Há também clientes Subversion disponíveis para download para o sistema operacional Windows. Um clientes desses é o TortoiseSVN
