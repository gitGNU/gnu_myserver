FastCGI

O FastCGI  é uma extensão ao CGI, não-dependente de linguagem, e que provê alta performance para aplicações, pois não usa APIs específicas de servidor. No FastCGI, você não tem um processo separado para cada script requisitado.

Você pode usar FastCGI no MyServer, bastando setar o MIME Type para sua linguagem de script para rodar como um FastCGI (configure como RUNFASTCGI).

Leia a seção sobre PHP sobre como usar PHP+FastCGI.

Servidores FastCGI Remotos

O MyServer tem a capacidade de usar um servidor FastCGI remoto. O seguinte MIME usa uma conexão remota:
<MIMETYPE>
<EXT>php</EXT>
<MIME>text/html</MIME>
<CMD>RUNFASTCGI</CMD>

<MANAGER>@host:port</MANAGER>
</MIMETYPE>
A definição para um gerenciador remoto deve começar com @.
Se você está configurando os MIME types pela aplicação Control Center, você pode adicionar um servidor remotoadicionando a linha abaixo no formulário
@HOST:PORT no campo de caminho do FastCGI
