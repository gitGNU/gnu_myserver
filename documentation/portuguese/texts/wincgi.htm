WinCGI

O WinCGI lhe permite ter comunicação com o novo processo através de arquivos .ini. O WinCGI só é suportado em ambiente Windows.

Para habilitar WinCGI no MyServer, você precisa configurar o  MIME Type correspondente aos seus arquivos da aplicação WinCGI, e fazê-os rodar como WinCGI (EXECUTEWINCGI)

Como no CGI, você pode colocar todas as suas aplicações num diretório cgi-bin dentro do diretório web no MyServer, como visto abaixo:



A pasta cgi-bin está diretamente abaixo da pasta web selecionada.



Contudo, aplicações WinCGI podem ser colocadas em qualquer lugar no diretório web.
