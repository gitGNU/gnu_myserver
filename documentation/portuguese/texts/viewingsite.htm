VISUALIZANDO SEU WEB SITE


Uma vez tendo suas páginas web gravadas na pasta correta, você vai querer vê-las, tanto na máquina em que elas estão gravadas, quanto na Internet.

Para ver seus arquivos em sua máquina local, digite:

http://localhost

na barra de endereços de seu browser.

Para ver seus arquivos na Internet, você pode digitar o IP de seu computador em qualquer browser. Por exemplo, se seu endereço IP for 36.65.34.32 seus arquivos podem ser encontrados em:

http://36.65.34.32


Usando um nome DNS

Se você preferir http://william.homeip.net ao invés de seu endereço IP, você tem que conseguir um nome de DNS.


Um nome DNS é um mecanismo de rede que associa um endereço IP com um nome mnêmonico, como www.google.com. Você pode conseguir um nome DNS usando um sistema grátis como o DynDNS.org. Esse serviço é para PCs com conexões sempre online (ADSL, cabo,...) que podem ser acessados por um bom pedaço do dia.

Se você não tem um endereço IP estático, seu endereço muda a cada vez que você se conecta na Internet. Mas, com um nome de DNS da DynDNS.org, você está acessível tanto se seu IP for dinâmico quanto se for estático.

Para você se certificar que seu nome de DNS se refere à sua máquina a cada vez que você se conectar na Internet, você deve também instalar um cliente de DNS dinâmico. Você pode pegar um cliente DNS grátis no endereço http://www.sitedevelopers.com.


Se você não gostar no nome http://myName.homeip.net mas preferiria ter um nome como http://www.myName.net você precisa registrar um nome de domínio em um sistema pago.
