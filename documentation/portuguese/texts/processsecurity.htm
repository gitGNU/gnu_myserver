SEGURANÇA DO PROCESSO


Em plataformas UNIX-like, é possível mudar o identificador de usuário e de grupo par ao processo MyServer após ele haver feito bind nas portas necessárias (só o root pode fazer bind em portas < 1024).

Identificador do usuário do processo

Usar uma linha como a seguinte no arquivo myserver.xml vai fazer o MyServer trocar seu identificador de usuário para o especificado:
<PROCESS_USER_ID>PID</PROCESS_USER_ID>
onde PID é o identificador a usar.

Identificador de grupo do processo

Usar uma linha como a seguinte no arquivo myserver.xml vai fazer o MyServer trocar seu identificador de grupo para o especificado:
<PROCESS_GROUP_ID>GID</PROCESS_GROUP_ID>
onde GID é o identificador de grupo a usar.

Atenção: Quando uma dessas features é usada, o autoreboot não será usado pelo MyServer, pois ele não poderá pegar as permissões antigas de volta.
