PLIKI HTML


HTML jest używany do tworzenia statycznych stron WWW a także w połączeniu z innymi językami do tworzenia dynamicznych stron WWW. Każdy może tworzyć strony WWW w HTML-u (Hypertext Markup Language), które później mogą być udostępniane w sieci Internet z wykorzystaniem serwera MyServer.


Najprostszą metodą utworzenia pliku HTML jest użycie edytora tekstu (np. Open Office lub Microsoft Word) i zapisanie dokumentu jako plik z rozszerzenie .html.  Możesz także pisać kod HTML korzystając z prostszych edytorów tekstów takich jak notatnik lub GNU Emacs.


Domyślnym miejsce gdzie umieszcza się pliki HTML jest katalog web.


Domyślny katalog 'web'.


Możesz użyć innego katalogu odpowiednio konfigurując wirtualny serwer.
