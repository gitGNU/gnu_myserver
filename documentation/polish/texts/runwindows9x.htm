URUCHAMIANIE MYSERVER-a

Microsoft Windows

Możesz uruchomić MyServer dwoma sposobami, które zostaną opisane poniżej:

1. Możesz uruchomić serwer wydając polecenie myserver.exe z menu Start
(działa dla systemów Windows 98, XP i każdego systemu bazującego na Windows NT)

a. Otwórz menu Start

b. Wybierz Uruchom

c. Wpisz poleceni C:\MyServer\myserver.exe(zamień ścieżkę C:\MyServer\ na odpowiednią dla twojej instalacji MyServer-a)


Uruchamianie aplikacji z menu Start


2. Jeżeli uruchamiasz MyServer na systemach z rodziny Windows NT (NT, 2000, XP, .NET) masz możliwość dostępu do aplikacji korzystając z Control Center i uruchomienie procesu serwera jako serwisu systemowego. Aby uzyskać dodatkowe informację w jaki sposób wykorzystać tę możliwość zapoznaj się z informacjami zawartymi w rozdziale Windows NT/URUCHAMIANIE JAKO SERWIS.
