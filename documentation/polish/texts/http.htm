HTTP

HTTP (Hypertext Transfer Protocol) jest zbiorem zasad (protokołem) który zarządza przesyłaniem plików w sieci Internet. Mogą to być pliki tekstowe, graficzne, video lub dźwiękowe. HTTP działa w oparciu o architekturę klient - serwer. Kiedy klient wysyła żądanie o dane, serwer przetwarza żądanie i odsyła do klienta odpowiednie dane. Protokół HTTP działa w warstwie 7 modelu ISO-OSI.

Protokół HTTP jest nieformalnym standardem przesyłania dokumentów w sieci Internet.

Aktualna wersja HTTP/1.1 protokołu HTTP opisana jest w dokumencie RFC 2616. Dodatkowe informacje możesz znaleźć na stronie Konsorcjum WWW.
