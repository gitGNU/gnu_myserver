ISAPI


ISAPI (Internet Server Application Program Interface) jest protokołem używanym w systemie Windows umożliwiającym serwerowi WWW na dynamiczne uruchamianie i wczytywanie modułów. Aby korzystać z ISAPI, musisz utworzyć plik DLL (dynamic link library) który udostępnia określone metody wywoływane przez serwer WWW.

Ostrzeżenie: Wystąpienie błędu w aplikacji ISAPI może spowodować przerwanie wykonywania procesu wywołującego, w tym przypadku serwera MyServer.
