ISAPI

ISAPI (Internet Server Application Program Interface) è un protocollo usato sui sistemi Windows dai web server per avere applicazioni CGI attraverso librerie esterne dinamiche(DLL). script. Per usare ISAPI hai bisogno di una libreria DLL che esporta alcune funzioni che verranno poi richiamate dal web server per comunicare con la libreria.
La libreria è caricata direttamente nell'area di memorie del webserver e risulta più veloce della tradizionale CGI.
Attenzione: Se una applicazione ISAPI si blocca essa può causare anche il blocco del webserver.
