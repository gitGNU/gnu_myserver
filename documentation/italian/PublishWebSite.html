PUBBLICA IL SITO WEB


Per dafault MyServer è installato con un solo virtual host accessibile da tutti gli IP. I files nella cartella web sono servono solo per una dimostrazione. Per usare i tuoi files tu devi sovrascrivere quelli nella cartella web con i tuoi

1. Cancella i files nella cartella web.
2. Aggiungi i tuoi files.

Aggiungi un file "default.html" essendo questo il primo file che gli utenti vedranno accedendo al tuo sito.

La cartella "web" è dove tu dovrai fare questi cambiamenti.


Se voui cambiare la cartella di default o settaggi relativi agli IP da usare devi configurare il file relativo ai virtual hosts.
