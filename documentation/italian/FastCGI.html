FastCGI

FastCGI è una estensione a CGI, not dipendente dal linguaggio usato, e permette alte prestazioni per le applicazioni, not usando lo stesso spazio di indirizzamento del web server. Non si ha bisogno di un processo separato per ogni chiamata a FastCGI come succede con CGI.

Per usarlo configura il tipo MIME per il tuo linguaggio per essere eseguito come FastCGI, mettendo RUNFASTCGI).

Leggi la sezione PHP su come usare PHP+FASTCGI.

Come per la CGI, puoi mettere le tue applicazioni nella sottodirectory cgi-bin della directory web directory in MyServer come visualizzato qui sotto:



La directory cgi-bin fa parte della directory web.



In ogni modo i files FastCGI possono essere messi da qualsiasi parte essendo mappati al web server attraverso la loro estensione.
