Proiectul MyServer

www.myserverproject.net



MyServer este un server de web puternic conceput pentru a fi rulat fără dificultăţi pe un calculator personal de către un utilizator având cunoştiinţe medii în domeniu.
Este bazat pe o arhitectura multi-thread lucru care-i permite să fie extrem de scalabil putând fi astfel utilizat atât pentru site-uri de dimensiuni mari cât şi pentru site-uri cu dimensiuni reduse. Implementează principalele protocoale utilizate în domeniu: HTTP, HTTPS, CGI, ISAPI, WinCGI şi FastCGI.
Puteti publica fişiere dvs. in câteva minute.

MyServer este disponibil atât pentru Windows cât şi pentru siteme de operare Unix-like.

Ţelul principal al serverului MyServer este de a creea o aplicaţie free şi puternică care să permită oricui să transforme calculatorul personal intr-un server de web şi să publice uşor fişierele dorite.
MyServer poate fi totodată folosit şi în mod profesional datorită arhitecturii sale multi-thread şi suportului său multi procesor.
