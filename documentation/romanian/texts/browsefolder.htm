TRAVERSAREA DIRECTOARELOR

Când un client cere server-ului un director care nu conţine nici un fişier implicit (ca de ex. index.html sau default.html), acesta din urmă indexează directorul şi trimite o lista cu fişierele din director clientului. Astfel clientul poate trece in revistă (browse) conţinutul directorului şi accesa orice fişier din cele afişate.

Imaginea de mai sus reprezintă pagina cu indexul unui director fără fişier implicit afişată de server pentru a fi traversat (răsfoit) de către client


Dacă doriţi să definiţi un alt mod de afişare a paginii cu indexul directorului puteţi modifica fişierul myserver.xml.

Localizaţi rândul: <BROWSEFOLDER_CSS>system/folderstyle.css</BROWSEFOLDER_CSS>

şi înclocuiţi system/folderstyle.css cu numele fişierului .css al dvs.

Fişierul system/folderstyle.css are următoarele valori implicite:

body { font-family: Arial; color: #000000;background-color:#FFFFFF }

Urmarea symlink-urilor

Cănd rulează pe sisteme de operare Unix-like (de ex.: Linux, *BSD, GNU/HURD, Solaris, etc.) MyServer poate urma symlink-uri.
Puteţi adăuga această facilitate fiecărei gazde virtuale din fişierul virtualhosts.xml sau modificând în fişierul myserver.xml pentru toate gazdele.
<FOLLOW_LINKS>YES</FOLLOW_LINKS>

Valoarea implicită este: <FOLLOW_LINKS>NO</FOLLOW_LINKS> Aceasta este valoarea fără nici o modificare de configurare cu care vine MyServer.
