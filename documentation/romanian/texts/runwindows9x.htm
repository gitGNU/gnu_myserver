PORNIREA MYSERVER

Microsoft Windows

Puteţi porni MyServer în două moduri aşa cum vom explica mai jos:

1. Puteţi rula myserver.exe de la meniul start
(funcţionează pentru Windows 98, XP,  sau oricare Windows bazat pe nucleul NT)

a. Deschideţi Meniul Start

b. Click pe Run

c. Scrieţi C:\MyServer\myserver.exe(schimbaţi C:\MyServer\ cu directorul unde este instalat MyServer la dvs. pe calculator)


Executarea programului de la meniul de start


2. Dacă rulaţi MyServer pe un Windows NT(NT, 2000, XP, .NET) puteţi controla programul folosind Control Center şi îl puteţi rula ca Serviciu. Pentru detalii despre cum să-l rulaţi ca serviciu consultaţi pagina RULEAZĂ CA SERVICIU .
