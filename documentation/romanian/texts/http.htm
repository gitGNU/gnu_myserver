HTTP

HTTP (Hypertext Transfer Protocol) este un set de reguli (un protocol) care guvernează transferul de fişiere pe Internet. Aceste fişiere pot avea formate diferite: text, grafic, sunet sau video. HTTP foloseşte o interfaţă simplă server-client, în care clientul formulează o cerere pentru un fişier sau o applicaţie iar serverul procesează cererea şi returnează datele cerute clientului. Protocolul HTTP lucrează la nivelul 7 al stivei ISO-OSI.

Protocolul HTTP este standardul 'de facto' pentru transferul documentelor HTML pe internet.

Versiunea HTTP curentă este HTTP/1.1 descrisă în RFC 2616. Mai multe informaţii pot fi găsite la WWW Consortium.
