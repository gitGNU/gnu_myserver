CUM SE FOLOSEŞTE PHP CU MYSERVER


PHP (Hypertext Preprocessor) este cel mai des folosit limbaj de script pe partea de server din lume. Este un limbaj de script folosit pentru a creea pagini cu conţinut dinamic. PHP poate fi folosit cu MyServer fie ca un executabil CGI sau ca server FastCGI. Când PHP este folosit ca executabil CGI MyServer va creea câte un nou proces pentru fiecare pagină solicitată.  Când este folosit ca server FastCGI MyServer refoloseşte aceelaşi process îmbunătăţind astfel performanţa. Pentru mai multe informaţii folosiţi siteul oficial FastCGI

Pentru a folosi PHP downloadaţi programul de instalare de la: www.php.net.

La ultimele versiuni trebuie modificat fişierul php.ini opţiunea cgi.force_redirect trebuie să fie zero. Căutaţi rândul:

;cgi.force_redirect= 0

în fişierul de configurare php.ini şi ştergeţi punct şi virgula de la începutul rândului.
După modificare acesta va fi:

cgi.force_redirect = 0
