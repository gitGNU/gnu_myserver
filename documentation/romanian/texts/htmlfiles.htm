FIŞIERE HTML


HTML este folosit pentru a scrie pagini de web statice şi e utilizat în conjuncţie cu alte limbaje pentru a creea pagini dinamice. Oricine poate crea fişiere HTML (Hypertext Markup Language) care pot fi vizualizate cu ajutorul MyServer.


Cel mai uşor mod de a scrie fişiere HTML este folosind un procesor de texte cum ar fi Open Office sau Microsoft Word. Fişierul astfel creat se salvează cu extensia .html .  De asemenea fişierele HTML pot fi scrise folosind un editor de texte cum ar fi notepad or GNU Emacs.


Locul implicit de salvare a fişierelor HTML este directorul web.


Locul implicit directorul "web" este selectat.


Se poate defini alt director configurânt virtual hosts.
