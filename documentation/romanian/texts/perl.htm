CUM SE FOLOSEŞTE PERL CU MYSERVER


Perl este un alt limbaj de script utilizat pe scară largă pentru manipularea textului şi pentru managementul bazelor de date de pe web. Perl se poate folosi eficient împreuna cu HTML şi XML ca şi cu alte limbaje de markup. Împreună cu modulul CGI.pm inclus cu distribuţia standard, formurile HTML sunt mai uşor de tratat.

Pentru a folosi Perl, pe calculatorul respectiv trebuie să existe un interpretor pentru Perl.

Pe sistemele gen UNIX aflaţi unde exista acest program folosind comanda:

type perl


Pentru Windows, probabil că trebuie instalat mai întâi.  Interpretorul şi documentaţia sunt disponibile la adresa Active State Tool Company.

Mai multe informaţii şi mostre de cod pot fi găsite pe siteul Perl ( www.perl.org) sau citind documentaţia inclusa cu interpretorul.
