PUBLICAREA UNUI SITE DE WEB


În mod implicit MyServer este instalat cu o singura gazdă accesibilă pentru poate adresele IP ale calculatorului respectiv. Fişierele incluse în versiunea downloadabilă sunt demostrative. Pentru a publica siteul dvs., fişierele din directorul web vor trebui modificate sau înlocuite.

1. Ştergeţi fişierele din directorul web,
2. Adăugaţi propriile dvs fişiere.
Asiguraţi-vă că aveţi un fişier denumit "default.html" pentru că acesta este prima pagina pe care o vor vedea vizitatorii care vor accesa siteul dvs.

Directorul "web" evidenţiat este locul unde veţi face modificările necesare.

Dacă doriţi să schimbaţi setările directorului implicit sau setarile IP modificaţi gazdele virtuale.
