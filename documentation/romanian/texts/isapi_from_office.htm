ISAPI


ISAPI (Internet Server Application Program Interface) este un protocol folosit pe calculatoarele care rulează Windows pentru a încărca şi rula o aplicaţie aflată într-un modul extern în spaţiu de adrese al serverului de web. Pentru a folosi ISAPI, trebuie creat un dll (dynamic link library) care să exporte anumiţi simboli care vor fi ulterior apelaţi/folosiţi de către serverul de web.

Atenţie: Dacă o aplicaţie ISAPI crapă aplicaţia care o apelează poate crăpa de asemenea din această cauză aşadar MyServer poate crăpa din cauza acestor aplicaţii.
