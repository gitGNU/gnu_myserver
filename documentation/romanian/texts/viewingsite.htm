VIZUALIZAREA SITEULUI DVS. DE WEB


Odată ce aveţi paginile dvs. de web în directorul corespunzător puteţi să le vizualizaţi atât de pe calculatorul unde sunt salvate cât şi de pe Internet.

Pentru a le vedea local scrieţi:

http://localhost

în locul pentru adresa a browserului dvs.

Pentru a le vedea de pe Internet, puteţi scrie adresa IP a calculatorului dvs. în locul pentru adresa a browserului. De exemplu dacă adresa dvs. de IP este 36.65.34.32 fişierele dvs. se pot gasi la:

http://36.65.34.32.

Utilizarea unui nume DNS

Probabil că preferaţi http://popescu.adresaip.net în locul adresei dvs. de IP. Pentru asta aveţi nevoie de un nume DNS.

DNS este un mecanism(un serviciu) care asociaza unui nume o adresa IP ca spre exemplu www.google.com. Puteţi obţine un nume DNS gratuit de aici: DynDNS.org. Serviciu DNS este de regulă disponibil pentru PC-urile conectate în permanenţă(ADSL, cable,...) şi care pot fi accesate oricând.

Dacă nu dispuneţi de o adresa IP statică, adresa dvs. IP se schimbă de fiecare dată cand vă conectaţi la Internet. Cu nume DNS de la DynDNS.org sunteţi accesibil indiferent dacă aveţi adresa IP statică sau dinamică.

Pentru a fi sigur că numele DNS refera adresa curentă a calculatorului dvs. de fiecare dată când vă conectaţi la Internet, trebuie să vă instalaţi un client de DNS dinamic. Puteţi să obţineţi un client DNS gratuit aici: http://www.directupdate.net.

Dacă nu agreaţi numele http://popescu.adresaip.net şi doriţi mai degrabă un nume precum http://www.popescu.net aveţi nevoie să înregistraţi acest domeniu prin intermediul unui registrar.
