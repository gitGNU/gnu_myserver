MYSERVER'ı BAŞLATMAK

Microsoft Windows

MyServer'ı aşağıda ele alacağımız iki yolda çalıştırabilirsiniz:

1. Başlat menüsündeki çalıştır'dan myserver.exe komutunu çalıştırabilirsiniz
(bu Windows 98, XP,  veya herhangi bir NT-tabanlı Windows'da çalışacaktır)

a. Başlat Menüsünü açın

b. Çalıştır'a basın

c. C:\MyServer\myserver.exe yazın (C:\MyServer\ adresini kendi MyServer kurulum adresiniz ile değiştirin)


Programı Başlat Menüsünden Çalıştırmak


2. Eğer MyServer'ı Windows NT ailesinde (NT, 2000, XP, .NET) çalıştırıyorsanız, programa ayrıca Denetim Masası üzerinden erişip Servis olarak çalıştırabilirsiniz. Bunu gerçekleştirmek için açıklamalar için, SERVİS OLARAK ÇALIŞTIRMA sayfasına danışın.
