MYSERVER ALTINDA NASIL PERL KULLANILIR


Perl, metin manipulasyonunda ve web veritabanı yönetiminde mükemmel yaygın olarak kullanılan başka bir komut dilidir. Perl ayıca HTML ve XML gibi diğer biçimleme dilleri ile de çalışır. Perl'in, standart dağıtımda bulunan, CGI.pm modülü ile HTML formlar kolaylıkla işlenebilir.

Perl'i kullanmak için, ilk önce bir yorumlayıcınızın olduğundan emin olmalısınız.

UNIX türevi sistemlerde, perl'in nerede olduğunu bulmak için, şu komutu kullanın:

type perl


Windows altında Perl kullanmak için, ilk önce yüklemelisiniz.  Yorumlayıcı ve belgeleri Active State Araç Şirketi
adresinde bulabilirsiniz.
Daha fazla bilgi ve kod örneklerine Perl web sitesi ( www.perl.org) veya yorumlayıcı ile birlikte gelen belgelere bakabilirsiniz.
