HTTP

HTTP (Hipermetin Transfer Protokolu) Internet üzerinden dosya aktarımı yöneten kurallar(protokol) setidir. Bu dosyalar metin, grafik, ses veya video olabilir. HTTP, istemcinin dosya veya uygulama talep ettiği ve sunucunun talebi işleyip doğru veriyi istemciye geri gönderdiği basit bir sunucu-istemci arayüzü kullanır. HTTP protokolü ISO-OSI yığınının 7. seviyesinde çalışır.

HTTP protokolü internet üzerinden belge aktarımı için asıl standarttır.

Mevcut HTTP sürümü RFC 2616'de tanımlanan HTTP/1.1 dir. Ayrıntılı bilgi WWW Consortium adresinde bulunabilir.
HTTP myserver'da kullanılan varsayılan protokoldür, veri düz metin olarak gönderilir, eğer şifreler gibi önemli verileri aktarmanız gerekiyorsa bir HTTPS sanal hostu kurmayı düşünmelisiniz.
Varsayılan myserver kurulumu, standart port 80 üzerinde bir HTTP sanal hostu ayarlar, UNIX-türevi sistemlerde yalnızca root kullanıcısı bu portu kullanma hakkına sahiptir, myserver ı çalıştırmadan veya varsayılan portu değiştirmeden önce root olduğunuzdan emin olun.
