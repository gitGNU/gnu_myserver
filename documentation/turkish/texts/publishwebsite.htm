WEB SİTESİ YAYINLAMAK


Ön tanımlı olarak, myServer tüm yerel makina IPlerinden erişilebilen tek ana makina ile kurulur. Sunucunun indirilen sürümünde yer alan dosyalar yalnızca tanıtım amaçlıdır. Dosyalarınızı başkaları ile paylaşmak için, web dizinindeki dosyaları kendi dosyalarınız ile değiştirmelisiniz.

1. Web dizinindeki mevcut dosyaları silin
2. Kendi dosyalarınızı ekleyin.
Kullanıcılar sunucuya eriştiğinde ilk göreceği dosya olduğundan, "default.htm" dosyasını koymayı unutmayın.

İşaretli "web" dizini, değişiklikleri yapacağınız yerdir

Eğer varsayılan dizin ayarlarını veya IP ayarlarını değiştirmek isterseniz, sanal ana makinaları ayarlamalısınız.
