WINCGI


WinCGI .ini dosyaları üzerinden yeni işlemler ile iletişim sağlar. WinCGI yalnızca Microsoft Windows platformu üzerinde desteklenir.
MyServer'da WinCGI'ı etkinleştirmek için, sizin WinCGI uygulamanıza uygun doğru MIME tiplerini seçmeniz, ve onları WinCGI (EXECUTEWINCGI) olarak çalışmaya ayarlamanız gerekir

CGI'da olduğu gibi , tüm uygulamalarınızı aşağıda görüldüğü gibi, MyServer'ın web dizini altında cgi-bin dizinine yerleştirebilirsiniz:


cgi-bin dizini, doğrudan seçili web dizini altındadır.



Buna rağmen, WinCGI uygulamaları web dizininde herhangi bir yere konabilir.
