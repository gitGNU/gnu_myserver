CONFIGURATION AVANCÉE


Vous trouverez ici d'autres options modifiables dans votre fichier de configuration de MyServer.

Types MIME pour les hôtes virtuels:


Dans le fichier myserver.xml, vous pouvez autoriser vos hôtes virtuels à avoir leurs propres types MIME ou non, en utilisant la ligne ci dessous, avec l'option "YES" (oui) ou "NO" (non).

<ALLOW_VHOST_MIME>YES</ALLOW_VHOST_MIME>

Fichiers MSCGI:

Vous pouvez autoriser l'utilisation des fichiers MSCGI dans MyServer, en définissant la valeur ci dessous dans votre fichier myserver.xml:

<ALLOW_MSCGI>YES</ALLOW_MSCGI>

Seuil GZIP:

Vous pouvez configurer MyServer pour qu'il compresse les données envoyées à l'utilisateur au format Gzip, si les données dépassent une certaine taille en octets, en ajoutant la ligne suivante dans le fichier myserver.xml:

<GZIP_THRESHOLD>1024</GZIP_THRESHOLD>



Ici, toute donnée supérieure à 1024 octets (1 Ko) sera automatiquement compressée avant d'être envoyée au client.

Permission d'accès des gestionnaires de données HTTP:


L'option <ALLOW_CGI>NO</ALLOW_CGI> empêche l'accès au gestionnaire de données CGI sur tout votre serveur. Cette option peut être ajoutée à chaque hôte virtuel, via le fichier virtualhosts.xml, ignorant ainsi la valeur du fichier de configuration global. ALLOW_CGI peut aussi apparaître comme ALLOW_FASTCGI, ALLOW_ISAPI, ALLOW_MSCGI, ALLOW_WINCGI, ALLOW_SEND_LINK, ALLOW_EXTERNAL_COMMANDS, ALLOW_SEND_FILE, chacun d'entre eux empêchant/autorisant l'accès au gestionnaire de données HTTP à utiliser.
