VOIR VOTRE SITE WEB


Une fois vos pages web enregistrés dans les bons répertoires, vous apprécierez de pouvoir y accéder en local comme sur Internet.

Pour voir les fichiers sur la machine locale, tapez:

http://localhost

dans la barre d'adresse de votre navigateur.

Pour voir vos fichiers depuis Internet, vous pouvez taper l'adresse IP de votre ordinateur depuis n'importe quel navigateur. Par exemple, si votre adresse IP est 36.65.34.32, vos fichiers seront accessibles depuis:

http://36.65.34.32 .

Utiliser un nom DNS

Si vous préférez http://william.homeip.net à la place de votre adresse IP, vous devez obten$ir un nom DNS.

Un nom DNS est un mécanisme réseau qui attribue à une adresse IP un nom, comme par exemple www.google.com. Vous pouvez obtenir un nom DNS grâce à des systèmes gratuits comme DynDNS.org. Ce service est pour des PCs connectés à Internet (ADSL, Cable, ...) pouvant être accessibles durant une grande partie de la journée.

Si vous ne possédez pas d'adresse IP statique (c'est à dire si votre adresse IP change à chaque reconnexion), vous pouvez quand même utiliser un nom de domaine comme DynDNS.org.

Afin d'être assuré que votre nom DNS fait bien référence à votre machine à chaque connexion sur Internet, vous devez installer un client DNS dynamique. Voici un des nombreux clients gratuits disponibles sur Internet : http://www.directupdate.net.

Si le nom http://myName.homeip.net ne vous satisfait pas et que vous préférez http://www.myName.net, vous devrez passer par un fournisseur de nom de domaine.
