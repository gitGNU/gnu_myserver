NAVIGATION DE FICHIERS

Quand un client demande l'accès à un répertoire du serveur et que celui ci ne trouve pas de page par défaut à afficher, il affiche le contenu du répertoire et envoie cette liste au client. Cela permet alors au client de naviguer entre les fichiers et répertoires du serveur.

Indexed folder for Browsing


Si vous voulez définir une feuille de style (CSS) pour le navigateur de fichiers, il suffit de modifier le fichier myserver.xml.

Repérez cette ligne: <BROWSEFOLDER_CSS>system/folderstyle.css</BROWSEFOLDER_CSS>

et remplacez system/folderstyle.css avec le nom de votre fichier .css.

Le fichier system/folderstyle.css posède le contenu suivant:

body { font-family: Arial; color: #000000;background-color:#FFFFFF }

Suivre les liens symboliques

Sur des systèmes d'exploitation (i.e.: Linux, *BSD, GNU/HURD, Solaris, etc.), MyServer peut suivre les liens symboliques.
Vous pouvez ajouter cette option pour chaque hôte virtuel en modifiant le fichier virtualhosts.xml, ou pour tout les hôtes en modifiant myserver.xml. <FOLLOW_LINKS>YES</FOLLOW_LINKS>

La valeur utilisée par défaut dans MyServer est: <FOLLOW_LINKS>NO</FOLLOW_LINKS>
