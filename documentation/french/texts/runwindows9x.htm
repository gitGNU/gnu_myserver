DEMARRER MYSERVER

Microsoft Windows

Vous pouvez lancer MyServer de deux manières différentes que voici:

1. Vous pouvez lancer la commande myserver.exe à partir du menu Démarrer.
(cela fonctionne pour Window 98, XP, ou tout windows de la famille NT)

a. Ouvrir le menu Démarrer

b. Cliquer sur Exécuter

c. Taper C:\MyServer\myserver.exe (changez C:\MyServer\ par votre chemin d'installation)


Lancement du programme à partir du menu Démarrer


2. Si vous exécutez MyServer sur un Windows de la famille NT (NT, 2000, XP, .NET), vous pouvez aussi accéder au Panneau de Configuration et le lancer comme service. Pour faire cela, consultez la page LANCER EN TANT QUE SERVICE.
