LANCER COMME UN SERVICE

Windows de la famille NT

Enregistrer le service sur le Système d'Exploitation


Si vous utilisez un Windows de la famille NT(NT, 2000, XP, .NET), vous pouvez lancer MyServer comme un Service. Avant de lancer le service, vous devez tout d'abord l'enregistrer en tant que tel sur le système.

1. Ouvrir le Centre de Controle de MyServer depuis le menu Démarrer

2. Cliquer sur Installer/Supprimer le Service dans la barre d'outils.

3. Cliquer sur Enregistrer le Service.



Enregistrement de Service

Note: Si vous voulez supprimer le Service plus tard, vous pourrez en cliquant sur Supprimer le service. Cela ne désinstallera pas le programme, mais évitera le démarrage du programme au lancement du système.


Démmarage et Arrêt du Service

Une fois le Service enregistré sur le système, vous pouvez le démarrer ou l'arrêter facilement depuis le Centre de Controle.


1. Sous le menu Fichier, choisir Exécuter en tant que Service pour démarrer le programme.

2. Lorsque vous voulez arrêter le programme, choisissez Arrêter le service dans le menu Fichier.



Démarrage et Arrêt du Service

Attention Afin de pouvoir fermer le Centre de Controle, choisissez la commande Quitter. Le serveur web et le centre de Controle sont deux applications distinctes. Fermer le Centre de Controle n'affecte pas le serveur web.

Pour plus d'informations sur les services, visitez le site de Microsoft sur www.microsoft.com et cherchez le terme services.
