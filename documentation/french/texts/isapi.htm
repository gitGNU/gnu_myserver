ISAPI


ISAPI (Internet Server Application Program Interface) est un protocole utilisé par les ordinateurs basés sur Windows pour lancer des applications dynamiques en chargeant un module externe dans l'espace d'adressage du processus du serveur web. Afin d'utiliser ISAPI, vous devez créer un DLL (Dynamic Link Library) qui permet l'export des symboles appelés par le serveur web.

Attention: Si une application ISAPI plante, cela pourrait causer le plantage de MyServer.


Pour plus d'informations, vous pouvez lire l'introduction ISAPI de Microsoft.
