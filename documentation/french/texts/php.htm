COMMENT UTILISER PHP AVEC MYSERVER


PHP (Hypertext Preprocessor) est le langage de script orienté serveur le plus utilisé au travers le monde. Un langage de script orienté serveur est utilisé pour créer des pages web avec de contenu dynamique. Vous pouvez utiliser PHP dans MyServer soit avec un executable CGI standard, ou alors avec un serveur FastCGI. Avec un CGI standard, MyServer créera un nouveau processus à chaque page interrogée. Avec un FastCGI, MyServer n'a pas besoin d'un nouveau processus, améliorant ainsi les performances. Réferrez vous à la documentation de FastCGI pour plus d'informations.

Afin d'utiliser PHP, vous devez tout d'abord télécharger l'archive d'installation sur : www.php.net.

Dans les dernières version de PHP, vous devrez également modifier le fichier php.ini pour mettre l'option cgi.force_redirect à null. Pour ce faire, trouvez la ligne suivante:

;cgi.force_redirect= 0

et supprimez le point virgule.
La ligne vaudra donc:


cgi.force_redirect = 0
