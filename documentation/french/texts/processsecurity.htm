SECURITÉ PROCESSUS


Sur des plates formes de type UNIX, il est possible de modifier l'identité et le groupe de l'utilisateur du processus de MyServer une fois les ports d'écoute démarrés (seul le root peut écouter les ports < 1024).

Identifiant utilisateur du processus

En utilisant la ligne ci dessous dans votre fichier myserver.xml, MyServer sera lancé avec l'utilisateur spécifié:
<PROCESS_USER_ID>PID</PROCESS_USER_ID>
où PID est l'identifiant à utiliser.

Identifiant groupe du processus

En utilisant la ligne ci dessous dans votre fichier myserver.xml, MyServer sera lancé avec le groupe spécifié:
<PROCESS_GROUP_ID>GID</PROCESS_GROUP_ID>
où GID est le groupe à utiliser.

Attention: L'utilisation d'une de ces options empêchera l'autoredémarrage de MyServer, car le retour aux anciennes permissions sera impossible.
