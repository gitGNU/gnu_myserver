SECURITE PROCESSUS


Sur des plates formes de type UNIX, il est possible demodifier l'identité et le groupe de l'utilisateur du processus de MyServer uen fois les ports d'écoute démarrés (seul le root peut écouter les ports < 1024).

Identifiant utilisateur du processus

En utilisant la ligne ci dessous dans votre fichier myserver.xml, MyServer changera pour l'utilisateur spécifié:
<PROCESS_USER_ID>PID</PROCESS_USER_ID>
où PID est l'identifiant à utiliser.

Identifiant groupe du processus

En utilisant la ligne ci dessous dans votre fichier myserver.xml, MyServer changera pour le groupe spécifié:
<PROCESS_GROUP_ID>GID</PROCESS_GROUP_ID>
où GID est le groupe à utiliser.

Attention: L'utilisation d'une de ces options empêchera l'autoreboot dans MyServer, car le retour aux anciennes permissions seront impossibles.
