HTTP

HTTP (Hypertext Transfer Protocol) est un ensemble de règles (protocoles) qui sègne sur le transfet de fichiers sur Internet. Ces fichiers peuvent être du texte, des images, du son ou de la vidéo. HTTP utilise une simple interface client-serveur, où le client demande l'accès à un fichier ou un programmer et le serveur traite les données et retourne les bonnes informations au client. Le protocle HTTP fonctionne sur la couche 7 du modèle ISO-OSI.

Le protocole HTTP est le standard de fait pour le transfert de documents sur internet.

La version actuelle d'HTTP est HTTP/1.1 décrit par le RFC2616. Plus d'informations peuvent être trouvées sur le site de WWW Consortium.
