HTTP

HTTP (Hypertext Transfer Protocol) est un ensemble de règles (protocoles) qui règne sur le transfet de fichiers sur Internet. Ces fichiers peuvent être du texte, des images, du son ou de la vidéo. HTTP utilise une simple interface client-serveur, où le client demande l'accès à un fichier ou un programmer et le serveur traite les données et retourne les bonnes informations au client. Le protocole HTTP fonctionne sur la couche 7 du modèle ISO-OSI.

Le protocole HTTP est le standard de fait pour le transfert de documents sur internet.

La version actuelle d'HTTP est HTTP/1.1 décrit par le RFC2616. Plus d'informations peuvent être trouvées sur le site de WWW Consortium.
HTTP est le protocole par défaut utilisé par MyServer. Les données sont envoyées en texte brut. Si vous devez envoyer des données confidentielles comme des mots de passes, alors pensez à l'utilisateur d'un hote virtuel HTTPS.
L'installation par dé de MyServer configure l'hote HTTP pour utiliser le port standard 80. Sous des systèmes UNIX, seul l'utilisateur root a les droits d'utilisation de ce port. Pensez donc à vérifier que vous êtes bien root avant de lancer MyServer ou de modifier le port.
