Subversion


MyServer est en constant développement. Ainsi, si vous voulez avoir les derniers fichiers, vous devrez utiliser le système Subversion. Les fichiers disponible sur le serveur sont souvent plus complets que les fichiers disponibles en téléchargement. Cependant, les fichiers sont testés avant d'être proposés au public, ce qui fait qu'ils sont plus stables que les fichiers disponibles sur le serveur. Nous vous recommandons donc l'utilisation des sorties officieles pour un environnement de production.

Téléchargement d'un client Subversion

Si vous ne possédez pas de client Subversion, visitez le site http://subversion.tigris.org/.

Accès anonyme

Pour obtenir les dernière sources disponibles, utilisez le client en tapant une des lignes suivantes:


1.  La première fois que vous copiez le contenu du serveur svn, tapez:

svn co https://svn.sourceforge.net/svnroot/myserverweb/trunk/myserverweb myserverweb

2. Si vous voulez mettre à jour vos fichiers, tapez depuis votre répertoire web de MyServer:

svn update

Il existe des clients Subversion pour le système d'exploitation Windows. Un de ceux ci est le client Tortoise SVN.
