COMMENT UTILISER PERL AVEC MYSERVER


Perl est un autre langage de script largement utilisé, idéal pour la manipulation de fichier et l'audit de bases de données web. Perl fonctionne également avec HTML et XML ou d'autres langages de balises. Avec le module CGI.pm de Perl, inclus dans la distribution de base, les formulaires HTML sont facilement manipulés.

Afin d'utiliser Perl, vous devez d'abord installer l'interpréteur.

Pour trouver l'emplacement de perl sur des systèmes basés sur UNIX, utilisez la commande:

type perl


L'interpréteur et la documentation de Perl pour Windows peuvent être trouvés sur le site de la Compagnie Active State Tool.

Plus d'informations et des exemples de code peuvent être trouvés sur le site web de Perl ( www.perl.org) ou en lisant la documentation inclue avec l'interpréteur.
