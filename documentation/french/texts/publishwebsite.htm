PUBLIER UN SITE WEB


Par défaut, MyServer est installé avec un seul hôte accessible depuis n'importe quelle interface de la machine. Les fichiers inclus dans la version téléchargeable du serveur n'ont qu'un but démonstratif. Afin de partager vos fichiers avec d'autres personnes, vous devrez remplacer les fichiers par défaut du répertoire web par les votres.

1. Supprimer les fichiers du répertoire web
2. Ajouter vos propre fichiers
N'oubliez pas d'ajouter un fichier nommé "default.html" qui sera la première page que verra l'utilisateur accédant à votre serveur.

Le répertoire "web" sélectionné est l'emplacement où vous effectuerez ces changements.

Si vous voulez modifier le répertoire de départ ou les paramètres IP, vous devez configurer les hôtes virtuels.
