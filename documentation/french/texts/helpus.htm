AIDEZ NOUS

Si vous êtes programmeur et que vous êtes intéressé par ce projet, vous pouvez nous aider.
Vous pouvez vérifier la liste TODO sur le site du projet www.myserverproject.net.
Si vous êtes intéressez, merci de contacter le responsable du projet.


Vous pouvez également aider si vous n'êtes pas programmeur:


Traduire MyServer dans d'autres langues,
Améliorer la documentation,
Tester et retourner les bogues,
Ecrire des exemples de scripts en Perl, PHP et autres CGI


Pour retourner de bogues nouveaux, veuillez utiliser le système de gestion de bogues de Sourceforge.

Si vous désirez faire un don pour ce projet, allez sur le site du projet et cliquez sur le bouton Donation.

Si vous avez des questions ou des commentaires, laissez un message sur le forum
